# 판데아 TRPG AGM v1.21

AGM은 `Artificial Game Master`다.  
진행자가 없을 때 장면을 열고, NPC를 움직이고, 비밀을 천천히 드러내고, 세계를 계속 굴리는 별도 진행 엔진이다.

이 파일은 사람이 직접 굴리는 `테이블용 AGM`이다. 종이, 주사위, 메모장만 있어도 쓸 수 있게 설계했다.  
솔로 플레이를 빠르게 굴리기 위한 도구지만, 둘이서 할 때도 진행자 없는 장면 판정 보조로 쓸 수 있다.  
장부, 비밀, 감정선, 장면 압력까지 세밀하게 관리하는 GM급 진행은 [DAGM](판데아_TRPG_DAGM_v1.21.md)을 쓴다.

## 빠른 카드: 실제 플레이 1페이지

플레이 중에는 이 카드만 펼쳐도 된다.  
막히면 `장면 -> 판정 -> 후속 -> 여파` 순서로 처리한다.

```text
1. 장면을 한 문장으로 적는다.
2. 필요한 카드 1개만 고른다.
3. 결과를 장면 문장으로 말한다.
4. 누가 기억하는지, 무엇이 남는지 적는다.
5. 선택을 다시 플레이어에게 돌려준다.
```

### 장면 프레임

| d6 | 압력 | 장면 질문 |
|---:|---|---|
| 1 | 굶주림 | 누가 먹지 못했나? |
| 2 | 추적 | 누가 먼저 도착하나? |
| 3 | 거래 | 누가 값을 올리나? |
| 4 | 비밀 | 무엇이 기록과 다르나? |
| 5 | 시간 | 무엇이 곧 닫히나? |
| 6 | 세계반응 | 어떤 소문으로 남나? |

### 예/아니오

가능성 보정: 거의 불가능 -2, 어려움 -1, 반반 0, 그럴듯함 +1, 거의 확실 +2.  
`2d6+보정`을 굴린다.

| 결과 | 답 |
|---:|---|
| 3 이하 | 아니오, 그리고 강한 수 |
| 4~5 | 아니오, 하지만 작은 단서 |
| 6~8 | 예, 하지만 대가 |
| 9~10 | 예 |
| 11 이상 | 예, 그리고 기회 |

자동 진행을 쓰면 항상 후속 1개를 붙인다.

| d6 | 후속 |
|---:|---|
| 1 | 누가 봤나? |
| 2 | 누가 잘못 이해했나? |
| 3 | 무엇이 남았나? |
| 4 | 무엇이 따라오나? |
| 5 | 언제 터지나? |
| 6 | 누가 이득 보나? |

### NPC 의도

NPC를 만나면 먼저 굴린다.  
`성격 + 감정 + 기억 + 겉의도`를 한 줄로 만든다.

| d6 | 성격 | 감정 | 기억 | 겉의도 |
|---:|---|---|---|---|
| 1 | 보호자 | 겁먹음 | 낯선 위협 | 도움 요청 |
| 2 | 계산가 | 초조함 | 번호/별명 앎 | 거래 |
| 3 | 생존자 | 굶주림 | 같은 처지 | 식량 확보 |
| 4 | 이름욕심 | 분노 | 원한과 겹침 | 시험 |
| 5 | 신념형 | 죄책감 | 팔 정보 | 도망 |
| 6 | 복수자 | 침착함 | 과거를 앎 | 적의 |

### RP 받아치기

플레이어가 캐릭터 대사로 말하면 숫자부터 말하지 않는다.  
NPC가 먼저 대사로 받아치고, 그 다음 필요한 규칙만 짧게 붙인다.

친근하게 말해도 NPC가 무조건 호응하지는 않는다.  
성향, 현재 감정, 싫어하는 말, 플레이어 기억과 어긋나면 대사로 받아치되 밀어낸다.

RP 길이: 가벼운 RP는 `대사 1줄 + 몸짓 1개`, 보통 RP는 `대사 2~3줄 + 기억/선택`, 중요한 RP는 `대사 + 감정선 + 장부 갱신`까지 쓴다.  
감탄사와 말버릇은 장면당 1~2번만 강하게 쓰고, 나머지는 몸짓으로 표현한다.

| d6 | 반응 |
|---:|---|
| 1 | 맞장구친다 |
| 2 | 농담으로 받아친다 |
| 3 | 더 세게 과장한다 |
| 4 | 진지하게 받아들인다 |
| 5 | 도전으로 받아친다 |
| 6 | 숨은 걱정을 농담 뒤에 숨긴다 |

| 태그 | 대사 방식 |
|---|---|
| 호탕함 | 크게 웃고, 몸짓이 크고, 칭찬도 싸움처럼 한다 |
| 건조함 | 짧게 말하고, 농담도 무표정하게 한다 |
| 겁먹음 | 웃어도 주변을 본다 |
| 계산적 | 농담 속에 조건을 섞는다 |
| 신념형 | 대사를 자기 원칙으로 돌린다 |
| 복수자 | 웃다가도 특정 이름에서 식는다 |

### 전투 압력

전투는 `적 AI -> 플레이어 선언 -> 판정 -> 압력 -> 여파`로 끝낸다.

| d6 | 압력 |
|---:|---|
| 1 | 누가 봤다 |
| 2 | 지형이 변했다 |
| 3 | 적이 배웠다 |
| 4 | 자원이 줄었다 |
| 5 | 소리가 났다 |
| 6 | 단서가 드러났다 |

### 정체와 반복 폭력

위험을 계속 피하면:

```text
멈춰 있는 동안 움직인 것은 ______다.
사라지는 기회는 ______다.
이제 선택을 요구하는 것은 ______다.
```

사람을 두 번 이상 반복해서 때리면:

```text
부서진 것은 ______다.
손에 남은 것은 ______다.
이 폭력을 기억하는 것은 ______다.
```

### 여파

장면 끝에는 하나만 적는다.

| d6 | 여파 |
|---:|---|
| 1 | 시계 +1 |
| 2 | 배고픔, 피로, 식량 압박 |
| 3 | NPC 태도 변화 |
| 4 | 소문 생성 |
| 5 | 기회 하나 사라짐 |
| 6 | 다음 장면 후크 생성 |

## 0. AGM과 DAGM의 차이

| 구분 | AGM | DAGM |
|---|---|---|
| 전체 이름 | Artificial Game Master | Detailed Artificial Game Master |
| 권장 환경 | 종이, 주사위, 사람 메모 | 상세 장부, 긴 메모, 컴퓨터/AI 권장 |
| 장점 | 빠르고 TRPG 테이블에 맞다 | 기억, 비밀, 감정선, 세계반응이 깊고 상대역 감각이 강하다 |
| 약점 | 품질이 사람 질문과 기록에 좌우된다 | 무겁고 사람이 손으로 돌리기 어렵다 |
| 쓰는 때 | 빠른 진행이 필요할 때 | 외로운 솔로 캠페인 감각이 필요할 때 |
| 비밀 처리 | 플레이어가 단서 장부를 직접 관리 | 상세 장부로 공개 정보와 숨은 정보를 분리 관리 |

## 1. AGM의 원칙

AGM은 플레이어의 정답을 대신 고르지 않는다.  
대신 아래 네 가지를 한다.

| 역할 | 하는 일 |
|---|---|
| 장면 연출 | 냄새, 소리, 사람, 물건, 압력을 보여준다 |
| 인물 구동 | NPC가 자기 성격, 감정, 기억, 목표대로 움직인다 |
| 비밀 관리 | 진실을 바로 말하지 않고 단서를 쌓는다 |
| 세계 반응 | 선택하지 않은 일도 시간, 소문, 시계로 진행한다 |

AGM의 금지:

- 플레이어 캐릭터의 행동을 대신 정하지 않는다.
- 숨은 진실을 단서 없이 바로 확정하지 않는다.
- 가장 효율적인 선택지를 알려주지 않는다.
- 실패를 장면 정지로 만들지 않는다.
- 성공을 흔적 없는 무결한 해결로 만들지 않는다.

## 2. 무게 선택

매 장면마다 전부 굴리면 느리다. 장면 무게를 먼저 정한다.

| 무게 | 언제 쓰나 | 굴리는 것 |
|---|---|---|
| 가벼움 | 이동, 짧은 조사, 사소한 대화 | 예/아니오 AGM + 후속 1개 |
| 보통 | NPC, 위험, 전투, 거래 | 장면 AGM + 해당 서브 AGM 1개 |
| 무거움 | 보스, 배신, 도시, 캠페인 갈림길 | 장면 AGM + 서브 AGM 2~3개 + 장부 갱신 |
| 최종장 | 엔딩, 신화의 대가, 최종 보스 | 모든 결과를 엔딩 질문과 연결 |

기본값은 `보통`이다.

AGM은 빠르지만 기록을 안 하면 깊이가 얇아진다.  
장면이 끝날 때 아래 3줄만은 반드시 적는다.

```text
누가 무엇을 원했나:
무엇을 기억하거나 오해했나:
그대로 두면 다음에 무엇이 움직이나:
```

질문이 떠오르지 않으면 아래 셋 중 하나만 고른다.

| 상황 | 자동 질문 |
|---|---|
| NPC가 밋밋함 | 이 NPC는 지금 무엇을 얻고 싶나? |
| 비밀이 사라질 듯함 | 플레이어가 볼 수 있는 단서는 무엇인가? |
| 장면이 끝났음 | 이 일은 누구의 기억이나 시계에 남나? |

## 3. AGM 기본 루프

장면이 멈추거나 판단이 필요하면 아래 순서로 처리한다.

```text
1. 장면 무게를 정한다.
2. 현재 입력을 한 문장으로 적는다.
3. 라우터에서 서브 AGM을 고른다.
4. 필요한 표를 굴린다.
5. AGM 답변 형식으로 말한다.
6. 장부를 갱신한다.
7. 플레이어에게 선택을 돌려준다.
```

현재 입력 예시:

- 굶주린 노예가 길을 막고 있다.
- 상자를 부쉈는데 안에서 음식이 나오지 않았다.
- 플레이어가 위험한 길을 세 번 피했다.
- 칠성 무기 경매 소문이 떴다.
- NPC가 도와주겠다고 하지만 눈을 피한다.

## 4. 서브 AGM 라우터

| 상황 | 돌릴 서브 AGM |
|---|---|
| 새 장면이 시작됨 | 장면 프레임 AGM |
| NPC가 등장함 | NPC 의도 AGM |
| 플레이어가 캐릭터 대사로 말함 | RP 받아치기 AGM |
| NPC를 믿을지 고민함 | NPC 진실 AGM |
| 대화가 건조함 | 감정/기억 AGM |
| 단서가 필요한데 정답을 말하면 안 됨 | 단서 사다리 AGM |
| 성공했지만 흔적이 필요함 | 결과/대가 AGM |
| 전투가 숫자 교환이 됨 | 전투 감독 AGM |
| 사람이 계속 맞거나 학살될 위험 | 반복 폭력 AGM |
| 플레이어가 아무것도 안 함 | 정체/위험 회피 AGM |
| 세계 이벤트가 붕 뜸 | 세계 연결 AGM |
| 캠페인이 방향을 잃음 | 주제/선택 AGM |
| 장면이 끝남 | 여파 AGM |

한 장면에는 보통 1개만 굴린다. 중요한 장면만 2~3개를 연결한다.

## 5. AGM 답변 형식

AGM은 표 결과를 그대로 읽지 않는다. 항상 장면 문장으로 바꾼다.

짧은 답:

```text
보이는 것:
느껴지는 것:
변하는 것:
지금 선택:
```

중요 답:

```text
결과:
클로즈업:
숨은 압력:
NPC 반응:
선택지:
후크:
```

선택지는 보통 2개, 중요한 장면은 3개다.

| 선택지 | 성격 |
|---|---|
| 안전한 선택 | 느리지만 피해가 적다 |
| 빠른 선택 | 시간을 벌지만 자원을 쓴다 |
| 위험한 선택 | 크게 얻을 수 있지만 흔적이 남는다 |

## 6. 장면 프레임 AGM

새 장면을 열 때 쓴다. d6을 세 번 굴린다.

```text
첫인상 -> 장면 압력 -> 장면 질문
```

| d6 | 첫인상 | 장면 압력 | 장면 질문 |
|---:|---|---|---|
| 1 | 냄새: 피, 비, 기름, 음식 | 굶주림 | 누가 먹지 못했나? |
| 2 | 소리: 사슬, 발소리, 기도 | 추적 | 누가 먼저 도착하나? |
| 3 | 사람: 기다림, 숨음, 판매 | 거래 | 누가 값을 올리나? |
| 4 | 물건: 문서, 장비, 낙인 | 비밀 | 무엇이 기록과 다르나? |
| 5 | 장소: 좁음, 추움, 더움 | 시간 | 무엇이 곧 닫히나? |
| 6 | 이상함: 너무 조용함, 너무 깨끗함 | 세계반응 | 이 장면이 어떤 소문으로 남나? |

기록 문장:

```text
이 장면은 ______로 시작하고, ______가 압박하며, 질문은 ______다.
```

## 7. 예/아니오 AGM

가능성을 정하고 2d6을 굴린다.

| 가능성 | 보정 |
|---|---:|
| 거의 불가능 | -2 |
| 어려움 | -1 |
| 반반 | 0 |
| 그럴듯함 | +1 |
| 거의 확실 | +2 |

| 2d6+보정 | 답 |
|---:|---|
| 3 이하 | 아니오. 그리고 강한 수 |
| 4~5 | 아니오. 하지만 작은 흔적 |
| 6~8 | 예. 하지만 대가, 지연, 오해 |
| 9~10 | 예 |
| 11 이상 | 예. 그리고 기회나 단서 |

자동 후속 질문:

| d6 | 후속 |
|---:|---|
| 1 | 누가 봤나? |
| 2 | 누가 잘못 이해했나? |
| 3 | 어떤 흔적이 남았나? |
| 4 | 무엇이 다음 장면까지 따라오나? |
| 5 | 언제 이 대가가 터지나? |
| 6 | 이 일로 누가 이득을 보나? |

## 8. NPC 의도 AGM

NPC를 만났을 때는 예/아니오를 묻기 전에 이 표를 먼저 쓴다. d6을 네 번 굴린다.

```text
성격 유형 -> 현재 감정 -> 플레이어에 대한 기억 -> 겉의도
```

| d6 | 성격 유형 | 현재 감정 | 플레이어에 대한 기억 | 겉의도 |
|---:|---|---|---|---|
| 1 | 보호자 | 겁먹음 | 낯선 위협으로 본다 | 도움을 원한다 |
| 2 | 계산가 | 초조함 | 콜로세움 번호나 별명을 안다 | 뭔가 숨기고 있다 |
| 3 | 생존자 | 굶주림 | 닮은 사람을 떠올린다 | 거래를 원한다 |
| 4 | 이름을 원하는 자 | 분노 | 원한의 대상과 겹쳐 본다 | 플레이어를 시험한다 |
| 5 | 신념형 | 죄책감 | 팔 수 있는 정보로 본다 | 도망치려 한다 |
| 6 | 복수자 | 이상한 침착함 | 같은 처지로 본다 | 적의를 품고 있다 |

중요 NPC는 숨은 압력도 굴린다.

| d6 | 숨은 압력 |
|---:|---|
| 1 | 누군가에게 빚졌거나 쫓기고 있다 |
| 2 | 먹여야 할 사람이나 지켜야 할 사람이 있다 |
| 3 | 문서, 열쇠, 도감 조각, 금줄 중 하나를 숨겼다 |
| 4 | 시간이 없다. 곧 경비, 마수, 상단, 사제가 온다 |
| 5 | 누군가에게 감시당하고 있다 |
| 6 | 이름, 신명, 주명, 콜로세움 번호 중 하나를 말할 수 없다 |

## 9. RP 받아치기 AGM

플레이어가 캐릭터 대사로 말하면 AGM은 먼저 캐릭터 대사로 받아친다.  
규칙 설명, 관계점, 판정 안내는 그 다음에 한두 줄만 붙인다.

단, 친근한 말은 자동 성공이 아니다.  
NPC는 자기 성격, 신념, 기억, 두려움, 싫어하는 말에 따라 호응하거나 밀어낸다.

```text
1. NPC 반응 방식을 굴린다.
2. NPC 말투 태그를 고른다.
3. NPC가 호응하는지, 밀어내는지 정한다.
4. 한두 문장의 대사로 먼저 답한다.
5. 몸짓, 표정, 건네는 물건 중 하나를 붙인다.
6. 필요한 경우에만 기억, 판정, 선택을 짧게 말한다.
```

RP 길이 단계:

| 단계 | 언제 쓰나 | 길이 |
|---|---|---|
| 가벼움 | 지나가는 농담, 짧은 맞장구 | 대사 1줄 + 몸짓 1개 |
| 보통 | 설득, 친분, 작은 갈등 | 대사 2~3줄 + 기억/선택 |
| 중요 | 고백, 배신, 신념 충돌, 관계 변화 | 대사 + 감정선 + 장부 갱신 |

감탄사와 말버릇은 장면당 1~2번만 강하게 쓴다.  
그 뒤에는 말버릇 대신 몸짓, 침묵, 시선, 손의 움직임으로 같은 성격을 표현한다.

| d6 | 반응 |
|---:|---|
| 1 | 맞장구친다 |
| 2 | 농담으로 받아친다 |
| 3 | 더 세게 과장한다 |
| 4 | 진지하게 받아들인다 |
| 5 | 도전으로 받아친다 |
| 6 | 숨은 걱정을 농담 뒤에 숨긴다 |

| 태그 | 대사 방식 |
|---|---|
| 호탕함 | 크게 웃고, 몸짓이 크고, 칭찬도 싸움처럼 한다 |
| 건조함 | 짧게 말하고, 농담도 무표정하게 한다 |
| 겁먹음 | 웃어도 주변을 본다 |
| 계산적 | 농담 속에 조건을 섞는다 |
| 신념형 | 대사를 자기 원칙으로 돌린다 |
| 복수자 | 웃다가도 특정 이름에서 식는다 |

성향이 어긋난 예시:

```text
플레이어: 하! 하! 하! 자네 나와 성격이 잘 맞는군!
AGM: "웃음은 좋다. 하지만 네 칼에 묻은 피까지 웃어넘길 생각은 없다."
그는 웃지 않고 네 칼끝을 내려다본다.
기억: 이 NPC는 너를 호탕한 사람이 아니라 위험을 농담으로 덮는 사람으로 기억한다.
선택: 사과한다, 피 묻은 이유를 설명한다, 농담을 거두지 않는다.
```

예시:

```text
플레이어: 하! 하! 하! 자네 나와 성격이 잘 맞는군!
AGM: "하! 하! 하! 내 귀가 아직 붙어 있는 걸 보니 우리 둘 다 꽤 튼튼하군!"
그는 네 등을 세게 두드리고 말린 고기를 반으로 찢어 건넨다.
기억: 이 동료는 너를 같이 크게 웃을 수 있는 사람으로 기억한다.
선택: 같이 떠든다, 사냥 이야기로 넘어간다, 힘겨루기를 건다.
```

## 10. 감정/기억 AGM

NPC가 생생하지 않을 때 쓴다. d6을 두 번 굴린다.

| d6 | 감정의 원인 | 플레이어에게 드러나는 방식 |
|---:|---|---|
| 1 | 굶주림 | 음식, 물, 냄새에 시선이 간다 |
| 2 | 빚 | 가격, 계약, 장부를 먼저 본다 |
| 3 | 죄책감 | 말이 늦고 눈을 피한다 |
| 4 | 원한 | 특정 이름에 과하게 반응한다 |
| 5 | 보호 본능 | 약자, 동료, 아이를 먼저 감싼다 |
| 6 | 야심 | 이름, 무기, 후원자 이야기에 밝아진다 |

| d6 | 기억의 형태 |
|---:|---|
| 1 | 플레이어와 닮은 사람을 기억한다 |
| 2 | 플레이어의 관중 별명을 들은 적 있다 |
| 3 | 같은 콜로세움 문서를 본 적 있다 |
| 4 | 플레이어가 한 폭력이나 선행의 소문을 안다 |
| 5 | 플레이어의 장비, 낙인, 말투를 기억한다 |
| 6 | 플레이어가 모르는 과거의 이름을 알고 있다 |

## 11. NPC 진실 AGM

NPC 비밀은 바로 확정하지 않는다. 아래 단서 사다리를 쓴다.

| 단계 | 얻는 것 | 예시 |
|---:|---|---|
| 1 | 이상한 감각 | 눈을 피한다, 손이 떨린다 |
| 2 | 반복 흔적 | 같은 이름을 빼고 말한다 |
| 3 | 누군가의 반응 | 다른 NPC가 그를 피한다 |
| 4 | 위험한 확인 | 문서, 열쇠, 피 묻은 물건 |
| 5 | 진실 확정 | 배신자, 조력자, 이용자, 희생자 |

큰 진실은 최소 단서 3개가 필요하다.  
단서 5단계에 도달하면 진실을 확정하고, NPC 진실 메모에 적는다.

## 12. 결과/대가 AGM

성공이 너무 깨끗하면 이 표를 쓴다. d6을 두 번 굴린다.

```text
대가가 남는 영역 -> 대가가 터지는 시점
```

| d6 | 영역 | 시점 |
|---:|---|---|
| 1 | 몸 | 즉시 |
| 2 | 장비 | 다음 행동 |
| 3 | 식량/돈 | 다음 장면 |
| 4 | NPC 관계 | 밤 또는 휴식 때 |
| 5 | 소문/평판 | 다음 도시 |
| 6 | 세계반응 시계 | 세션 끝 |

기록 문장:

```text
성공한다. 하지만 대가는 ______에 남고, ______에 터진다.
```

## 13. 전투 감독 AGM

전투는 질문을 많이 하지 않는다.

```text
적 AI -> 플레이어 선언 -> 판정 -> 기회/꼬임 -> 장면 압력 갱신
```

장면 압력:

| d6 | 압력 |
|---:|---|
| 1 | 누가 봤다 |
| 2 | 지형이 변했다 |
| 3 | 적이 배웠다 |
| 4 | 자원이 줄었다 |
| 5 | 소리가 났다 |
| 6 | 단서가 드러났다 |

전투가 숫자 교환이 되면 d6을 굴린다.

| d6 | 전투를 장면으로 바꾸는 질문 |
|---:|---|
| 1 | 이 싸움을 누가 보고 있나? |
| 2 | 적은 왜 아직 물러나지 않나? |
| 3 | 지형에서 움직이는 것은 무엇인가? |
| 4 | 어느 자원이 먼저 바닥나나? |
| 5 | 이 적을 살려두면 무엇을 알 수 있나? |
| 6 | 이 전투가 끝난 뒤 남는 이름은 무엇인가? |

## 14. 반복 폭력 AGM

사람에게 `친다`, `또 친다`, `아직 서 있어? 또 친다`가 반복되면 쓴다.

```text
부서진 것은 ______다.
손에 남은 것은 ______다.
이 폭력을 기억하는 것은 ______다.
```

| d6 | 결과 |
|---:|---|
| 1 | 큰 소리. 경비, 관중, 추적자, 마수 중 하나가 온다 |
| 2 | 잔해. 임시 무기, 방패, 엄폐물, 증거품 중 하나가 된다 |
| 3 | 흔적. 누가 했는지 보이고 다음 숨기기/변명 -1d |
| 4 | 안쪽 단서. 원한, 장부, 출구, 이름 중 하나가 나온다 |
| 5 | 원래 길이 막히고 위험한 길이 열린다 |
| 6 | 다친 사람. 적, NPC, 동료, 자기 중 하나가 피해를 본다 |

두 번째 반복부터 이 표를 먼저 쓰고 공격한다.  
세 번째 반복부터는 장면 압력도 1개 더 갱신한다.

## 15. 정체/위험 회피 AGM

압력이 있는데 두 번 연속 `쉰다`, `기다린다`, `아무것도 안 한다`, `위험한 건 피한다`가 나오면 쓴다.

```text
멈춰 있는 동안 움직인 것은 ______다.
사라지는 기회는 ______다.
이제 선택을 요구하는 것은 ______다.
```

| d6 | 결과 |
|---:|---|
| 1 | 시간이 지난다. 시계 하나 +1 |
| 2 | 몸이 버티지 못한다. 배고픔, 피로, 식량 중 하나 압박 |
| 3 | 누군가 먼저 움직인다 |
| 4 | 안전한 길이 닫힌다 |
| 5 | 보상이 줄거나 다른 손에 넘어간다 |
| 6 | 작은 사람이 도움을 요구한다 |

정체로 피해는 피할 수 있다.  
하지만 큰 단서, 희귀 보상, 자유점수, 보스 약점은 얻지 못한다.

## 16. 세계 연결 AGM

세계 이벤트가 붕 뜨면 아래 셋을 묻는다.

| 질문 | 답하는 법 |
|---|---|
| 누구에게 먼저 닿나? | 출신, 직업, 흉터, 도감, 장비가 관련 있는 PC |
| 누가 이득을 보나? | 세력, NPC, 상단, 종교, 추적자, 마수 |
| 당장 무엇이 나빠지나? | 가격, 길, 평판, 안전, 식량, 법, 날씨, 추적 |

기록 문장:

```text
이 사건은 ______에게 먼저 닿고, ______가 이득을 보며, 지금 당장 ______가 나빠진다.
```

## 20. 전체 AGM 순서

세션 시작:

```text
지난 장면의 여파 확인 -> 현재 시계 확인 -> 첫 장면 프레임 -> 플레이어 선택
```

장면 중:

```text
플레이어 행동 -> 필요한 서브 AGM 선택 -> 판정/표 -> 답변 형식 -> 장부 갱신
```

장면 끝:

```text
여파 AGM -> NPC/세계 시계 갱신 -> 다음 후크 1개
```

세션 끝:

```text
세계 이벤트 -> PC 연결 질문 3개 -> 장부 정리 -> 다음 세션 첫 냄새/소리/사람 정하기
```

## 21. 빠른 예시

상황: 굶주린 노예가 길을 막는다.

```text
서브 AGM: NPC 의도 AGM
성격: 생존자
감정: 굶주림
기억: 같은 처지로 봄
겉의도: 거래를 원함
숨은 압력: 먹여야 할 사람이 있음
```

AGM 답:

```text
보이는 것: 노예는 칼보다 네 식량 주머니를 먼저 본다.
느껴지는 것: 싸우려는 자세인데, 눈은 계속 뒤쪽 우리로 간다.
변하는 것: 뒤쪽에서 작은 기침 소리가 난다.
지금 선택: 밀어붙이거나, 식량을 걸고 거래하거나, 뒤쪽 우리를 확인한다.
```

상황: 플레이어가 아무것도 안 하고 쉰다를 반복한다.

```text
서브 AGM: 정체/위험 회피 AGM
멈춰 있는 동안 움직인 것: 노예 사냥꾼
사라지는 기회: 낮의 안전한 길
선택 요구: 밤길로 갈지, 숨어서 식량을 쓸지
```

상황: 칠성 무기 경매 소문이 떴다.

```text
서브 AGM: 세계 연결 AGM
누구에게 먼저 닿나: 가문 인질 출신 PC
누가 이득을 보나: 위조 문서 상단
무엇이 나빠지나: 도시 통행 검문
```

AGM 답:

```text
결과: 경매 소문은 네 옛 가문 문장과 함께 돈다.
클로즈업: 전단지 모서리에 네가 아는 붉은 실 매듭이 있다.
숨은 압력: 위조 문서 상단이 통행권 가격을 두 배로 올렸다.
NPC 반응: 동료는 그 무기가 네 누명과 연결됐다고 본다.
선택지: 경매장으로 가거나, 전단지 출처를 추적하거나, 통행권을 위조한다.
후크: 밤에 누군가 네 관중 별명을 부른다.
```
