# 판데아 TRPG DAGM v1.21

DAGM은 `Detailed Artificial Game Master`다.  
AGM보다 장면, 비밀, NPC 기억, 세계 시계, 감정선, 규칙 압력을 더 자세히 관리하는 상세형 진행 엔진이다.  
손으로도 굴릴 수 있지만 장부가 많아 무겁다. 그래서 긴 메모, 스프레드시트, 컴퓨터, AI를 쓰면 훨씬 편하다.

빠른 진행이 필요하면 AGM을 쓴다.  
외로운 솔로 캠페인에서 진짜 상대역이 필요하면 DAGM을 쓴다.  
DAGM은 속도보다 몰입을 우선한다.
여러 명이 함께 플레이할 때도 쓸 수 있지만, 핵심 기능은 솔로 플레이어가 혼자서 NPC 대답, 장기 기억, 세계 반응을 얻도록 돕는 데 맞춰져 있다.

목표는 단순하다.

```text
혼자 플레이해도 진짜 GM과 하는 느낌에 최대한 가까워진다.
표가 대답하는 것이 아니라, 세계와 NPC가 나에게 대답하게 만든다.
```

## 1. AGM과 DAGM

| 구분 | AGM | DAGM |
|---|---|---|
| 전체 이름 | Artificial Game Master | Detailed Artificial Game Master |
| 환경 | 종이, 주사위, 사람 메모 | 상세 장부, 긴 메모, 컴퓨터/AI 권장 |
| 우선순위 | 속도와 실전성 | 몰입과 상대역 감각 |
| 품질 | 질문과 기록 실력에 좌우 | 상태 추적과 서브 엔진으로 보강 |
| 비밀 | 플레이어가 단서로 관리 | 상세 장부로 숨은 정보와 공개 정보를 분리 |
| 권장 | 실제 TRPG 테이블, 빠른 진행 | 외로운 솔로 캠페인, 무진행자 장기 캠페인 |

DAGM은 플레이어를 이기려고 하지 않는다.  
또 플레이어의 선택을 대신하지 않는다.  
대신 세계가 멈추지 않게 하고, NPC가 기억하게 하고, 비밀이 너무 빨리 까이지 않게 한다.

## 2. DAGM의 상태 장부

DAGM은 아래 상태를 계속 유지한다.

```text
공개 장부:
  현재 장면:
  현재 장소:
  현재 시간:
  PC 상태:
  동료 상태:
  공개 NPC:
  공개 단서:
  공개 시계:
  플레이어가 아는 소문:

숨은 장부:
  NPC 진짜 목표:
  숨은 압력:
  비밀 단서 단계:
  보스 약점 단계:
  세계 이벤트 원인:
  추적자 계획:
  아직 공개하지 않은 대가:

연속성 장부:
  지난 장면 여파:
  플레이어가 반복한 행동:
  NPC가 기억하는 행동:
  사라진 기회:
  다음 장면 후크:
```

플레이어에게는 공개 장부만 보여준다.  
숨은 장부는 DAGM만 사용한다. 단, 단서가 3개 이상 쌓이면 일부를 공개 장부로 옮긴다.

## 3. DAGM 턴 순서

플레이어가 행동을 말하면 DAGM은 내부적으로 아래 순서를 따른다.

```text
1. 입력 분류
2. 관련 장부 호출
3. 서브 엔진 선택
4. 규칙 판정 필요 여부 결정
5. 결과 생성
6. 대가와 여파 배치
7. 공개 답변 작성
8. 장부 갱신
9. 행동 가능한 단서나 다음 선택 반환
```

플레이어에게는 7번과 필요한 공개 장부만 보여준다.

## 3-1. DAGM 밀도 조절 장치

DAGM은 품질이 높지만 무겁다.  
그래서 기능을 빼는 것이 아니라 `처리 밀도`를 조절한다.

짧은 DAGM, 보통 DAGM, 깊은 DAGM, 초고도화 DAGM은 기능 단계가 아니다.
모든 DAGM은 항상 아래 코어를 수행한다.

```text
입력 분류 -> 관련 장부 확인 -> 규칙 필요 여부 판단 -> NPC/세계 반응
-> 대가와 여파 배치 -> 공개 답변 -> 장부 갱신 -> 선택 반환
```

차이는 무엇을 생략하느냐가 아니라, 얼마나 자세히 보여주고 얼마나 많이 새 장부로 펼치느냐다.

| 밀도 | 쓸 때 | 반드시 하는 일 | 줄이는 것 |
|---|---|---|---|
| 짧은 DAGM | 이동, 배식, 짧은 조사, 잡담 | 코어 전체를 돌리고 장면, 기억, 선택을 남김 | 출력 1~3문장, 장부 갱신 1줄로 압축 |
| 보통 DAGM | NPC, 거래, 전투, 탈출 준비 | 코어 전체 + NPC 반응/규칙/대가/기억을 분리 | 숨은 장부와 연속성은 각 1줄까지만 새로 씀 |
| 깊은 DAGM | 배신, 보스, 도시 사건, 캠페인 갈림길 | 코어 전체 + 감정선, 숨은 압력, 세계 시계, 장기 여파를 모두 연결 | 장면 끝에 요약 정리 |
| 초고도화 DAGM | GM 상대역 감각이 꼭 필요한 핵심 장면 | 코어 전체 + 문장 구조 랜덤 + GM 수 + 행동 분류 + 장부 압축 | 길이를 늘리지 않고 말하는 순서와 반응 방식을 바꿈 |

짧은 DAGM도 숨은목표, 세계 시계, NPC 기억을 무시하지 않는다.
그 장면에 걸려 있으면 `후보`, `압력`, `기억 1줄`로 압축한다. 압축할 수 없을 만큼 중요해지면 보통, 깊게, 초고도화 중 맞는 밀도로 올린다.

아래 조건이 2개 이상이면 깊게 이상으로 올린다.

| 조건 | 깊게 이상으로 올릴 이유 |
|---|---|
| 이름 있는 NPC가 등장함 | 기억과 감정선이 남는다 |
| 배신, 약점, 숨은목표가 걸림 | 숨은 장부가 장면을 바꾼다 |
| 전투 결과가 장기 후폭풍을 만든다 | 부상, 원한, 소문이 남는다 |
| 세력, 도시, 세계 시계와 연결됨 | 세계가 따로 움직인다 |
| 플레이어가 중요한 관계 대사를 함 | NPC 카드와 감정선이 필요하다 |
| 보스, 엔딩, 신화의 대가가 걸림 | 장면 하나가 캠페인을 바꾼다 |

콜로세움 프롤로그에서는 밀도를 낮게 유지한다.

- 배식과 훈련은 짧은 DAGM으로 처리하되, NPC 기억과 장면 여파는 1줄 남긴다.
- 첫 경기와 밤의 거래는 보통 DAGM으로 처리한다.
- 깊은 DAGM은 배신, 죽음, 출구 방식 확정, 장기 후폭풍이 걸릴 때만 연다.
- 숨은 장부는 장면마다 새 항목 1개까지만 만든다. 나머지는 `미결 압력`으로 묶는다.
- 세계 시계는 출구 개방 직전 한 번만 갱신해도 된다.
- 필수 생존 체크 8개는 기존 장면에 2~3개씩 묶고, 심화 체크 5개는 자연스럽지 않으면 첫 바깥밤으로 넘긴다.

시간 제한:

```text
한 장면에서 새 장부 항목이 3개를 넘으면 나머지는 미결 압력으로 묶는다.
플레이어 선택지가 나온 순간 DAGM은 더 설명하지 않는다.
5분 이상 기록하고 있다면 다음 장면으로 넘긴다.
```

## 3-2. DAGM 출력 밀도 단계

DAGM은 항상 풀파워로 말하지 않는다.  
장면의 무게에 맞춰 출력 밀도를 고른다.

출력 밀도는 기능을 빼는 규칙이 아니다.  
내부 코어는 항상 돌리고, 플레이어에게 보이는 말과 장부 기록만 압축한다.

| 단계 | 쓰는 때 | 출력 | 장부 기록 |
|---|---|---|---|
| 얕게 | 이동, 배식, 짧은 농담, 사소한 조사 | 장면/결과/선택을 1~3문장으로 압축 | 공개 장부 1줄, 필요하면 숨은 압력 후보 1줄 |
| 보통 | NPC 대화, 거래, 첫 경기, 탈출 준비 | 장면 + NPC 반응 + 판정/대가 + 기억/선택 | 공개 장부 1~3줄, 숨은/연속성 각 1줄 |
| 깊게 | 배신 단서, 반복 폭력, 큰 전투, 세력 사건 | 장면 + 클로즈업 + 숨은 압력 단서 + 여파 + 선택 | 공개/숨은/연속성 갱신, 세계 시계 검토 |
| 초고도화 | 캠페인 갈림길, 보스, 엔딩, 비밀 NPC, 신화의 대가 | 결과 + 감정선 + 세계 반응 + 장기 대가 + GM 수 + 다음 장면 구조 | 전체 장부를 정리하되 공개 답변은 선택 질문 1개에서 멈춤 |

기본 출력은 `보통` 이하로 둔다.  
`깊게` 이상은 장면이 끝난 뒤에도 따라오는 결과가 있을 때만 쓴다.

출력 밀도를 낮추는 규칙:

- 같은 NPC와 짧게 주고받는 대화는 `얕게`로 처리한다.
- 판정이 필요 없는 장면은 `판정: 없음` 라벨을 생략할 수 있다.
- 플레이어가 이미 선택할 수 있으면 추가 설명을 멈춘다.
- 숨은 장부는 공개하지 말고, 공개 단서 1개나 압력 후보 1개로 장면에 섞는다.
- 생략하는 것은 라벨과 분량이지, NPC 기억, 대가, 세계 반응 자체가 아니다.

## 3-3. 짧은 답 모드

솔로 플레이의 기본은 `짧은 답 모드`다.  
DAGM을 장면마다 전부 열면 기록량이 늘고 플레이 속도가 죽는다.

짧은 답 모드는 기능을 빼는 규칙이 아니다.  
내부에서는 입력 분류, 규칙 심판, NPC/세계 반응, 대가, 기억, 선택 반환을 모두 처리하고, 화면에는 필요한 것만 남긴다.

짧은 답 모드를 기본으로 쓰는 장면:

| 장면 | 왜 짧게 처리하나 |
|---|---|
| 배식, 이동, 야영 준비 | 반복되기 쉽고 선택이 작다 |
| 짧은 조사, 냄새 맡기, 물건 확인 | 단서 1개면 충분하다 |
| 가벼운 RP, 농담, 인사 | 감정선은 남기되 장부를 열 필요는 적다 |
| 전투 중 일반 라운드 | 매 라운드 길어지면 전투가 멈춘다 |
| 밤 장면의 중간 행동 | 핵심 선택 전까지 기록량을 줄인다 |

짧은 답 모드 출력:

```text
장면/결과: 한두 문장
대가/기억: 있으면 1줄
선택: 무엇을 하나?
```

예:

```text
장면/결과: 배식통 바닥에서 탄내 나는 죽이 긁혀 나오고, 네 손목 번호를 본 배식 노예가 잠깐 숟가락을 멈춘다.
대가/기억: 그는 네 번호를 기억한다. 대신 지금은 말하지 않는다.
선택: 말을 건다, 죽을 챙긴다, 손목을 숨긴다.
```

전투 짧은 답:

```text
압력: 무엇이 불리해졌나?
결과: 피해/위치/상태 1줄
다음: 누가 무엇을 하나?
```

밤/비밀 장면은 재미있지만 길어진다.  
손목 번호, 숨은 낙인, 비밀 방문자, 거래 같은 장면은 처음부터 깊은 DAGM이나 초고도화로 열지 말고 아래 순서로 올린다.

```text
짧은 답 -> 단서 2개 쌓임 -> 보통 밀도 -> 선택이 캠페인을 바꾸면 전체 밀도
```

기록량 제한:

- 한 장면 공개 기록은 최대 3줄.
- 숨은 장부는 후보 1개만 새로 만든다.
- 같은 밤 장면에서 판정/대가가 3번을 넘으면 마지막 선택을 받고 장면을 닫는다.
- 전투는 라운드마다 장부를 쓰지 말고, 전투 끝에 부상, 원한, 소문만 정리한다.

## 3-4. 초고도화 DAGM

`초고도화`는 새 엔진이 아니다.
SDAGM처럼 이름을 하나 더 만들지 않고, DAGM 안에서 가장 GM답게 작동하는 운용 모드로 둔다.

초고도화 DAGM의 목표:

```text
표를 읽는 느낌을 없앤다.
문장 구조를 매번 바꾼다.
플레이어 행동을 먼저 분류하고, 판정이 필요한지 판단한다.
성공/실패를 장면 반응, NPC 기억, 세계 시계로 돌려준다.
공개 답변은 라벨을 붙이지 않고 가능한 한 하나의 GM 문장으로 말한다.
선택 질문 하나를 던진 순간 멈춘다.
```

초고도화는 장면마다 쓰지 않는다.
아래 중 2개 이상이 걸릴 때만 쓴다.

| 조건 | 이유 |
|---|---|
| 이름 있는 NPC가 핵심 반응을 한다 | 말투, 기억, 숨은목표가 중요하다 |
| 숨은 단서나 배신이 걸렸다 | 공개 정보와 숨은 정보를 분리해야 한다 |
| 보스, 엔딩, 신화의 대가가 걸렸다 | 장면 하나가 캠페인 구조를 바꾼다 |
| 플레이어가 큰 선택을 했다 | 세계가 기억해야 한다 |
| 같은 장면이 반복될 위험이 있다 | GM 수로 압력을 바꿔야 한다 |
| 혼자 하는데 상대역 감각이 필요하다 | 단순 표보다 대답받는 느낌이 중요하다 |

초고도화 절차:

```text
1. 문장 구조를 굴린다.
2. 장면 또는 행동을 분류한다.
3. 판정 필요 여부를 정한다.
4. 부드러운 GM 수 또는 강한 GM 수를 고른다.
5. 공개 답변을 쓴다.
6. 숨은 장부와 연속성 장부를 각각 최대 1줄만 갱신한다.
7. 선택 질문 하나를 던지고 멈춘다.
```

### 초고도화 문장 구조

초고도화 DAGM은 항상 같은 순서로 말하지 않는다.
d6을 하나 굴려 공개 답변의 첫머리를 정한다.
아래 형식은 내부 순서일 뿐이고, 실제 출력에서는 `장면:`, `압력:`, `단서:` 같은 라벨을 붙이지 않는다.

| d6 | 문장 구조 | 형식 |
|---:|---|---|
| 1 | 감각 선행 | 장면 -> 압력 -> 클로즈업 -> 반응 -> 질문 |
| 2 | NPC 선행 | 반응 -> 장면 -> 압력 -> 단서 -> 질문 |
| 3 | 압력 선행 | 압력 -> 장면 -> GM 수 -> 단서 -> 질문 |
| 4 | 단서 선행 | 단서 -> 장면 -> 반응 -> 압력 -> 질문 |
| 5 | 질문 선행 | 질문 -> 장면 -> 압력 -> GM 수 -> 남는 것 |
| 6 | 침묵 선행 | 침묵/정지 -> 클로즈업 -> 반응 -> 압력 -> 질문 |

이 표는 말투를 바꾸기 위한 표다.
기능을 더하는 표가 아니다. 내부에서는 항상 코어 전체를 처리한다.
초고도화 공개 답변은 보통 한 문장, 길어도 한 문단으로 끝낸다.

### 초고도화 행동 분류

플레이어가 행동을 말하면 초고도화 DAGM은 먼저 행동을 분류한다.

| 행동 | 기본 판정 후보 | GM처럼 보는 것 |
|---|---|---|
| 말한다, 설득한다, 속인다, 흥정한다 | 매력+설득/공연, 지성+기만 | 상대의 감정, 기억, 숨은목표 |
| 때린다, 막는다, 제압한다 | 근력/민첩+무기, 회피/막기/버티기 | 피해보다 위치, 소리, 목격자 |
| 숨는다, 도망간다, 잠입한다 | 민첩+은신/운동, 지성+판단 | 길과 흔적, 따라오는 사람 |
| 살핀다, 냄새 맡는다, 도감을 본다 | 지성+감각/도감 | 정답 대신 확인 가능한 단서 |
| 마법/율법/권능을 쓴다 | 마나+마법학 또는 율법 대가 | 장면을 바꾼 힘의 반동 |
| 기다린다, 쉰다, 숨죽인다 | 판정 없음 또는 체력/정신 | 안전 대신 움직이는 세계 시계 |

판정은 불확실하고 실패가 재밌을 때만 굴린다.
성공이 당연하거나 실패해도 아무 일도 없으면 판정하지 않는다.

### 초고도화 GM 수

초고도화는 선택지를 많이 주는 모드가 아니다.
대신 GM처럼 부드러운 수와 강한 수를 골라 장면을 움직인다.

| d6 | 부드러운 GM 수 | 강한 GM 수 |
|---:|---|---|
| 1 | 위험을 먼저 보여준다 | 위험이 지금 도착한다 |
| 2 | 대가를 예고한다 | 대가를 바로 치르게 한다 |
| 3 | NPC 의도를 행동으로 보인다 | NPC가 먼저 움직여 선택지를 좁힌다 |
| 4 | 닫히는 문과 열리는 틈을 함께 둔다 | 시간, 자원, 의심 중 하나를 깎는다 |
| 5 | 단서를 주고 해석은 맡긴다 | 숨은 압력 하나를 공개 단서로 끌어올린다 |
| 6 | 다음 판정의 위험을 보여준다 | 성공해도 지울 수 없는 흔적을 남긴다 |

2d6 결과가 `아니오` 쪽이면 강한 수를 쓴다.
`예` 쪽이면 부드러운 수를 쓰되, `예. 하지만`에서는 대가를 반드시 장면 안에 보인다.

### 초고도화 출력 예시

장면 열기:

```text
기록을 쥔 사람은 틀린 부분을 읽지 않고 손으로 가린 채 콜로세움 우리의 마른 종이 접히는 소리만 이상하게 크게 남기고, DAGM은 대가를 예고하되 아직 터뜨리지 않으며, 문서 모서리에 접힌 자국과 덜 바랜 안쪽이 보이니, 지금 판정을 걸고 들어가나, 단서 하나를 더 모으나?
```

행동 처리:

```text
손목 번호를 가리고 배식 담당의 표정을 살피는 행동은 조사로 읽히고 지성+감각/도감 판정이 맞으며, 원하는 결과는 얻지만 누군가 오해하고, 배식 담당은 네 번호를 말하지 않는 대신 다음 배식 때 먼저 네 손부터 볼 테니, 숨기나, 먼저 말하나, 아니면 상대가 착각하게 두나?
```

초고도화의 금지:

- 장면을 길게 설명하려고 초고도화를 쓰지 않는다.
- 라벨 목록으로 답하지 않는다.
- 숨은 진실을 단서 없이 바로 말하지 않는다.
- 플레이어 대신 최선의 행동을 고르지 않는다.
- 선택지가 나온 뒤에도 추가 설명을 이어가지 않는다.
- 한 장면에서 새 장부 항목을 3개 이상 만들지 않는다.

## 4. 입력 분류

| 플레이어 입력 | 분류 |
|---|---|
| 문을 연다, 상자를 부순다 | 물건/장치 행동 |
| NPC에게 말한다 | 대화 행동 |
| 때린다, 협박한다 | 전투/폭력 행동 |
| 도망간다, 숨는다 | 회피/추적 행동 |
| 기다린다, 쉰다 | 정체 행동 |
| 도감을 본다, 냄새를 맡는다 | 조사/단서 행동 |
| 마법을 쓴다 | 그라듀/마법 행동 |
| 율법을 쓴다 | 율법/대가 행동 |
| 어디로 갈지 정한다 | 이동/세계 행동 |
| 목표를 바꾼다 | 캠페인/주제 행동 |

입력이 애매하면 DAGM은 질문을 되묻지 않고 가장 가까운 분류를 고른다.  
단, 캐릭터 생사, 목표 포기, 동료 살해처럼 되돌리기 어려운 선택은 확인 질문 1개를 할 수 있다.

## 5. 서브 엔진

DAGM은 한 번에 여러 서브 엔진을 돌릴 수 있다.  
하지만 공개 답변은 짧게 유지한다.

| 엔진 | 담당 |
|---|---|
| 규칙 심판 | 판정 슬롯, 보상 소스, 키워드 소진, 방호, 피해 계산 |
| 장면 감독 | 첫인상, 압력, 행동 가능한 단서, 후크 |
| NPC 배우 | 성격, 감정, 기억, 말투, 호칭, 몸짓, 행동 |
| 비밀 관리자 | 단서 단계, 숨은 목표, 배신/약점 확정 |
| 세계 시뮬레이터 | 세계 이벤트, 세력, 도시, 시계 |
| 결과 편집자 | 예/아니오, 하지만, 그리고, 대가 배치 |
| 감정선 감독 | 동료 반응, 관계점, 죄책감, 이름, 두려움 |
| 전투 감독 | 적 AI, 장면 압력, 반복 폭력, 항복/제압 |
| 정체 압력 관리자 | 위험 회피, 기회 소실, 시간 경과 |
| 연속성 관리자 | 이전 사건, NPC 기억, 장비/단서/소문 유지 |

## 6. DAGM 출력 형식

DAGM은 플레이어가 말한 방식에 맞춰 답한다.  
플레이어가 캐릭터 대사로 말하면 DAGM은 반드시 먼저 캐릭터 대사로 받아친다. 숫자와 규칙은 그 다음에 짧게 붙인다.

RP 입력 처리:

```text
1. NPC가 먼저 캐릭터 대사로 답한다.
2. 몸짓, 표정, 건네는 물건, 침묵 중 하나를 붙인다.
3. 필요한 경우에만 판정, 기억, 선택을 짧게 붙인다.
4. 관계점, 호감, 의심 같은 숫자는 장면 문장 뒤에 둔다.
```

NPC 자율성:

- 플레이어가 친근하게 말해도 NPC가 무조건 호응하지 않는다.
- DAGM은 먼저 NPC의 성격, 현재 감정, 플레이어 기억, 싫어하는 말, 숨은목표를 확인한다.
- 성향이 맞으면 받아치며 가까워진다.
- 성향이 어긋나면 받아치되 밀어낸다.
- 호감, 관계점, 협조는 칭찬만으로 오르지 않는다. `위험 감수`, `실질적 도움`, `상대의 기억 존중`, `싫어하는 선을 넘지 않음` 중 하나가 있어야 한다.

RP 길이 단계:

| 단계 | 언제 쓰나 | 출력 길이 |
|---|---|---|
| 가벼운 RP | 농담, 인사, 짧은 반응 | 대사 1줄 + 몸짓 1개 |
| 보통 RP | 설득, 친분, 작은 갈등 | 대사 2~3줄 + 기억/선택 |
| 중요한 RP | 고백, 배신, 신념 충돌, 관계 변화 | 대사 + 감정선 + 장부 갱신 |

말투 과사용 방지:

- 감탄사, 말버릇, 호칭은 장면당 1~2번만 강하게 쓴다.
- 같은 느낌을 반복해야 하면 대사 대신 몸짓, 침묵, 시선, 손의 움직임으로 표현한다.
- 호탕한 NPC가 매번 웃지 않아도 된다. 웃으려다 멈추거나, 어깨를 치거나, 잔을 밀어주는 행동으로 호탕함을 보인다.

나쁜 출력:

```text
NPC는 당신에게 호감을 느낍니다. 관계점 +1 후보입니다.
```

좋은 출력:

```text
"하! 하! 하! 나도 같은 생각일세! 네 웃음소리만 들어도 술통 뚜껑이 먼저 도망가겠군!"

그는 네 등을 세게 두드리고, 자기 몫의 말린 고기를 반으로 찢어 건넨다.

기억: 이 동료는 너를 같이 크게 웃을 수 있는 사람으로 기억한다.
선택: 같이 마시며 떠든다, 바로 사냥 이야기로 넘어간다, 힘겨루기를 건다.
```

행동 선언, 조사, 전투처럼 대사가 아닌 입력에는 기본적으로 아래 형식으로 답한다.

```text
장면:
판정:
결과:
대가:
기억:
선택:
```

기본은 짧은 답 모드다.  
얕게 답할 때도 위 항목은 내부적으로 처리하지만, 화면에는 `장면/결과 + 대가/기억 1줄 + 선택`으로 압축한다.

판정이 필요 없으면 `판정: 없음`이라고 쓴다.  
중요 장면에서는 아래 형식을 쓴다.

```text
결과:
클로즈업:
숨은 압력:
NPC 반응:
세계 반응:
선택지:
장부 갱신:
```

숨은 장부의 진실은 그대로 쓰지 않는다.  
공개 가능한 단서만 `클로즈업`, `NPC 반응`, `세계 반응`에 섞는다.

## 7. 공개/숨은 정보 규칙

| 정보 종류 | 공개 여부 |
|---|---|
| 캐릭터가 직접 본 것 | 공개 |
| 냄새, 소리, 표정, 물건 | 공개 |
| NPC의 진짜 목적 | 숨김 |
| 보스의 최종 약점 | 숨김 |
| 도시의 핵심 비밀 | 숨김 |
| 세계 이벤트의 진짜 원인 | 숨김 |
| 단서 3개가 쌓인 추론 | 일부 공개 |
| 플레이어가 틀리게 해석한 정보 | 정정하지 말고 장면으로 반응 |

DAGM은 거짓말해도 된다.  
하지만 거짓말은 NPC의 말, 소문, 문서 오류, 착각으로만 한다.  
서술 자체가 플레이어를 속이면 안 된다.

## 8. 단서 확정 규칙

큰 진실은 단서 3개가 필요하다.

| 단서 수 | 처리 |
|---:|---|
| 1 | 이상한 감각 |
| 2 | 반복되는 흔적 |
| 3 | 그럴듯한 추론 가능 |
| 4 | 위험한 확인 |
| 5 | 진실 확정 |

예:

```text
NPC 배신:
1단서: 눈을 피함
2단서: 같은 이름을 빼고 말함
3단서: 다른 NPC가 그를 보고 숨음
4단서: 손에 콜로세움 장부 잉크
5단서: 장부 사본 발견
```

## 9. DAGM NPC 엔진

NPC가 등장하면 내부적으로 아래를 만든다.

```text
성격:
현재 감정:
플레이어 기억:
겉목표:
숨은목표:
두려움:
선 넘는 조건:
지금 가진 자원:
말투:
자주 쓰는 감탄사:
웃는 방식:
플레이어를 부르는 호칭:
좋아하는 농담:
싫어하는 말:
호응 한계:
친해졌을 때 하는 행동:
화났을 때 말투:
숨은 압력이 드러나는 순간:
장면 감탄사 사용:
```

NPC 카드는 한 번 만들면 유지한다.  
다음 장면에서 다시 만나면 성격을 새로 굴리지 않고, 현재 감정과 플레이어 기억만 갱신한다.

빠른 생성:

| d6 | 성격 | 감정 | 기억 | 겉목표 |
|---:|---|---|---|---|
| 1 | 보호자 | 겁먹음 | 낯선 위협 | 도움 요청 |
| 2 | 계산가 | 초조함 | 번호/별명 앎 | 거래 |
| 3 | 생존자 | 굶주림 | 같은 처지로 봄 | 식량 확보 |
| 4 | 이름욕심 | 분노 | 원한과 겹쳐 봄 | 시험 |
| 5 | 신념형 | 죄책감 | 팔 정보로 봄 | 도망 |
| 6 | 복수자 | 침착함 | 과거를 안다 | 적의 |

말투 태그:

| 태그 | 대사 방식 | 숨은 압력이 새는 방식 |
|---|---|---|
| 호탕함 | 크게 웃고, 몸짓이 크고, 칭찬도 싸움처럼 한다 | 웃음이 갑자기 멈춘다 |
| 건조함 | 짧게 말하고, 농담도 무표정하게 한다 | 대답이 더 짧아진다 |
| 겁먹음 | 웃어도 주변을 본다 | 문, 발소리, 그림자를 확인한다 |
| 계산적 | 농담 속에 조건을 섞는다 | 숫자, 빚, 대가 이야기가 튄다 |
| 신념형 | 대사를 자기 원칙으로 돌린다 | 율법, 신명, 죄 이야기가 나온다 |
| 복수자 | 웃다가도 특정 이름에서 식는다 | 손이 무기나 흉터로 간다 |

NPC 대사 생성 순서:

```text
1. 말투 태그를 적용한다.
2. 현재 감정을 한 단어만 섞는다.
3. 플레이어에 대한 기억을 호칭이나 농담으로 드러낸다.
4. 숨은목표는 직접 말하지 않고, 몸짓이나 피하는 단어로 흘린다.
5. 마지막에 플레이어가 받을 수 있는 선택을 남긴다.
```

짧은 NPC 카드 예시:

```text
NPC: 술고래 전 검투사 가론
성격: 보호자
현재 감정: 들뜸
플레이어 기억: 같이 크게 웃을 수 있는 사람
겉목표: 사냥 동료 구하기
숨은목표: 예전 동료의 시신을 찾기
두려움: 전투가 끝난 뒤 혼자 남는 것
말투: 호탕함
감탄사: 하! 하! 하!
웃는 방식: 목을 젖히고 크게 웃음
호칭: 큰웃음 친구
좋아하는 농담: 술, 힘, 사냥감 과장
싫어하는 말: 버리고 간다, 쓸모없다
호응 한계: 버림, 배신, 약자를 조롱하는 농담에는 웃지 않음
친해졌을 때 하는 행동: 자기 식량을 반으로 나눔
화났을 때 말투: 웃음이 끊기고 아주 낮게 말함
숨은 압력이 드러나는 순간: 죽은 동료 이야기가 나오면 잔을 내려놓음
장면 감탄사 사용: 0/2
```

NPC는 장면마다 자기 목표를 향해 1칸 움직인다.  
플레이어가 아무것도 하지 않아도 NPC는 멈추지 않는다.

NPC 자율성 규칙:

| 조건 | DAGM 처리 |
|---|---|
| 성격과 맞는 RP | 먼저 대사로 호응하고, 작은 기억을 남긴다 |
| 성격과 안 맞는 RP | 대사로 받아치되, 태도는 좁히거나 선을 긋는다 |
| 싫어하는 말이 나옴 | 즉시 분위기가 식고, 선택지에 사과/해명/강행을 둔다 |
| 숨은목표와 충돌 | 겉으로는 웃어도 몸짓이나 침묵으로 밀어낸다 |
| 반복 칭찬만 함 | 관계점은 오르지 않고, NPC가 조건이나 행동을 요구한다 |

밀어내는 대사 예시:

```text
플레이어: 하! 하! 하! 자네 나와 성격이 잘 맞는군!
신념형 NPC: "웃음은 좋다. 하지만 네 칼에 묻은 피까지 웃어넘길 생각은 없다."

그는 웃지 않는다. 시선은 네 얼굴보다 칼끝에 오래 머문다.

기억: 이 NPC는 너를 호탕한 사람이 아니라 위험을 농담으로 덮는 사람으로 기억한다.
선택: 사과한다, 피 묻은 이유를 설명한다, 농담을 밀어붙인다.
```

## 10. DAGM 장면 엔진

장면을 열 때 내부적으로 네 가지를 정한다.

```text
첫인상:
압력:
장면 질문:
닫히는 기회:
```

| d6 | 첫인상 | 압력 | 장면 질문 |
|---:|---|---|---|
| 1 | 냄새 | 굶주림 | 누가 먹지 못했나? |
| 2 | 소리 | 추적 | 누가 먼저 도착하나? |
| 3 | 사람 | 거래 | 누가 값을 올리나? |
| 4 | 물건 | 비밀 | 무엇이 기록과 다르나? |
| 5 | 장소 | 시간 | 무엇이 곧 닫히나? |
| 6 | 이상함 | 세계반응 | 어떤 소문으로 남나? |

### 10-1. 시드 해석법: 단어를 장면으로 바꾸기

DAGM 표에서 나온 단어는 출력문이 아니라 장면 재료다.  
사람이 종이로 굴릴 때는 단어를 그대로 읽지 말고, 아래 순서로 장면 안에 배치한다.

```text
1. 현재 장소에 놓는다.
2. 감각 하나로 시작한다.
3. 첫인상을 물건, 사람, 흔적, 이상한 점 중 하나로 바꾼다.
4. 압력은 누군가의 행동으로 보여준다.
5. 장면 질문은 직접 묻지 말고 이상한 변화나 틈으로 만든다.
6. 플레이어가 건드릴 수 있는 물건, 사람, 통로, 말실수를 하나 이상 남긴다.
```

나쁜 처리:

```text
문서와 장비와 낙인이 있고 거래가 일어난다. 누가 값을 올리나?
```

좋은 처리:

```text
죽통이 오기 전, 장부가 먼저 온다.
배식 담당은 네 얼굴보다 손목의 낙인을 먼저 확인한다.
귀족 하인은 녹슨 검 한 자루를 문서 위에 올려놓고 낮은 목소리로 값을 부른다.
그때 철창 밖 어둠 속에서 누군가가 한 닢을 더 얹는다.
장부 귀퉁이, 검자루, 하인의 흔들리는 손목이 모두 닿을 듯 가까이 있다.
```

선택지 목록은 필수가 아니다.  
초심자, 멈춘 플레이어, 복잡한 장면에는 2~3개를 보여줘도 된다. 기본 DAGM은 선택지를 나열하기보다 장면 안에 행동 가능한 단서를 심고 `무엇을 하나?`로 돌려준다.

압력은 이름으로 말하지 말고 장면 행동으로 보여준다.

| 압력 | 장면으로 바꾸는 법 |
|---|---|
| 굶주림 | 부족한 죽그릇, 비어 있는 손, 빼앗기는 순서, 식량을 건 거래 |
| 추적 | 먼저 도착한 발소리, 남은 흔적, 이름을 부르는 경비, 닫히는 도주로 |
| 거래 | 값, 조건, 빚, 장부, 누군가의 흥정, 사람과 물건이 같은 표에 오름 |
| 비밀 | 기록과 다른 이름, 숨긴 물건, 말하지 않는 손짓, 지워진 문장 |
| 시간 | 곧 닫히는 문, 교대 직전의 경비, 시작되는 경기, 사라지는 사람 |
| 세계반응 | 목격자, 붙는 별명, 퍼지는 소문, 다음 장면에 남는 기록 |

장면 질문도 그대로 읽지 않는다.

| 장면 질문 | 장면 안의 사건으로 바꾸기 |
|---|---|
| 누가 먹지 못했나? | 죽그릇 하나가 부족하다. 배식 담당은 그걸 실수라고 부르지 않는다. |
| 누가 먼저 도착하나? | 네가 움직이기 전에, 발소리가 이미 모퉁이 안쪽에서 멈춘다. |
| 누가 값을 올리나? | 어둠 속 누군가가 네 이름도 모른 채 판돈을 더 얹는다. |
| 무엇이 기록과 다르나? | 장부에는 네 이름이 없는데, 낙인 번호는 붉은 줄로 표시되어 있다. |
| 무엇이 곧 닫히나? | 철문 위 쇠고리가 내려가기 시작한다. 닫히면 오늘 밤까지 열리지 않는다. |
| 어떤 소문으로 남나? | 경비 하나가 네 행동을 보고 웃으며 새 별명을 중얼거린다. |

짧게 만들 때는 아래 한 줄 공식을 쓴다.

```text
감각 -> 이상한 물건/사람 -> 압력을 보여주는 행동 -> 건드릴 수 있는 틈 -> 무엇을 하나?
```

닫히는 기회는 장면이 끝날 때 사라지거나 비싸진다.

## 11. 결과 편집 엔진

판정 결과를 장면으로 바꾼다.

| 결과 | DAGM 처리 |
|---|---|
| 실패 | 막히지 않고 다른 길, 대가, 위험을 보여준다 |
| 성공 | 결과를 준다 |
| 예, 하지만 | 결과 + 대가 + 후속 질문 |
| 예, 그리고 | 결과 + 기회 + 새 후크 |
| 아니오, 하지만 | 실패 + 작은 단서 |
| 강한 수 | 피해, 추적, 시계, NPC 결정, 안전한 길 폐쇄 |

`예, 하지만`의 대가는 아래 중 하나다.

| d6 | 대가 |
|---:|---|
| 1 | 시간이 든다 |
| 2 | 소리가 난다 |
| 3 | 자원을 쓴다 |
| 4 | 누가 오해한다 |
| 5 | 일부만 된다 |
| 6 | 더 큰 문제의 단서가 보인다 |

## 12. 전투 감독 엔진

전투는 숫자 교환으로 만들지 않는다.

```text
적 AI -> 플레이어 선언 -> 규칙 판정 -> 장면 압력 -> 여파
```

장면 압력:

| d6 | 압력 |
|---:|---|
| 1 | 누가 봤다 |
| 2 | 지형이 변했다 |
| 3 | 적이 배웠다 |
| 4 | 자원이 줄었다 |
| 5 | 소리가 났다 |
| 6 | 단서가 드러났다 |

사람 대상 반복 폭력:

```text
부서진 것은 ______다.
손에 남은 것은 ______다.
이 폭력을 기억하는 것은 ______다.
```

두 번째 반복부터 적용한다.  
세 번째 반복부터는 장면 압력도 1개 추가한다.

## 13. 정체 압력 엔진

플레이어가 위험을 계속 피하면 세계가 움직인다.

```text
멈춰 있는 동안 움직인 것은 ______다.
사라지는 기회는 ______다.
이제 선택을 요구하는 것은 ______다.
```

| d6 | 결과 |
|---:|---|
| 1 | 시계 +1 |
| 2 | 배고픔, 피로, 식량 압박 |
| 3 | 누군가 먼저 움직임 |
| 4 | 안전한 길 폐쇄 |
| 5 | 보상 감소 |
| 6 | 작은 사람이 도움 요청 |

## 14. 세계 시뮬레이터

세션 끝이 아니어도 세계는 움직인다.

| 트리거 | 처리 |
|---|---|
| 하루가 지남 | 지역 시계 1개 +1 |
| 큰 소문 발생 | 관련 세력 목표 1칸 전진 |
| 플레이어가 위험을 회피 | 기회 하나 사라짐 |
| 보스가 살아남음 | 다음 등장 때 대응책 1개 추가 |
| NPC를 살림 | 관계망 확장 |
| NPC를 죽임 | 기억하는 사람 또는 문서 생성 |

세계 이벤트가 뜨면 반드시 연결한다.

```text
누구에게 먼저 닿나?
누가 이득을 보나?
당장 무엇이 나빠지나?
```

## 15. 감정선 엔진

DAGM은 동료와 NPC 감정선을 자동으로 흔든다.

| 트리거 | 감정선 반응 |
|---|---|
| 식량을 나눔 | 보호적 NPC 호감 +1 |
| 사람을 반복 폭행 | 동료가 두려움/오해/반대 중 하나 |
| 위험 회피 반복 | 야심적 NPC가 실망하거나 먼저 움직임 |
| 도감 기록 | 도감기록자, 마법사, 학자 반응 |
| 이름을 버림 | 신명, 주명, 관중 별명 관련 후크 |
| 동료를 구함 | 관계점 +1 또는 숨은 기억 공개 |

감정선은 보상이 아니라 방향이다.  
숫자보다 다음 장면에서 누가 어떤 말을 하는지를 먼저 만든다.

## 16. AGM과 DAGM 처리 비교 예시

AGM은 사람이 표 1~2개를 굴려 빠르게 장면을 이어주는 방식이다.  
DAGM은 같은 입력을 받으면 공개 장부, 숨은 장부, NPC 기억, 세계 시계까지 함께 갱신한다. 그래서 느리지만 GM에 더 가깝다.

### 예시 0: RP로 농담을 건다

입력:

```text
플레이어: 하! 하! 하! 자네 나와 성격이 잘 맞는군, 하! 하! 하!
```

| 구분 | 처리 |
|---|---|
| AGM | RP 받아치기 AGM을 굴린다. 반응과 말투 태그를 골라 짧은 대사와 기억 1개만 붙인다. |
| DAGM | NPC 카드의 말투, 감탄사, 호칭, 좋아하는 농담, 현재 감정, 플레이어 기억을 함께 사용해 먼저 대사로 답한다. |

DAGM 공개 답변:

```text
"하! 하! 하! 나도 같은 생각일세, 큰웃음 친구! 네 웃음소리만 들어도 술통 뚜껑이 먼저 도망가겠군!"

가론은 네 등을 세게 두드리고, 자기 몫의 말린 고기를 반으로 찢어 건넨다. 웃고 있지만, '버리고 간다'는 말이 나오자마자 손가락이 잠깐 굳는다.
다음 말에서는 다시 크게 웃지 않고, 대신 네 어깨를 주먹으로 툭 친다.

기억: 가론은 너를 같이 크게 웃을 수 있는 사람으로 기억한다.
선택: 같이 마시며 떠든다, 바로 사냥 이야기로 넘어간다, 왜 버려지는 말을 싫어하는지 묻는다.
```

내부 갱신:

```text
플레이어 기억: 같이 크게 웃을 수 있는 사람.
감정선: 가론 호감 후보 +1.
숨은 압력 단서: 버려지는 말에 반응함.
장면 감탄사 사용: 1/2.
```

### 예시 1: 굶주린 노예에게 "또 친다"

입력:

```text
굶주린 노예가 길을 막는다.
플레이어: 말 많다. 친다. 아직 서 있어? 또 친다.
```

| 구분 | 처리 |
|---|---|
| AGM | 반복 폭력 카드를 굴린다. 부서진 것은 신뢰, 손에 남은 것은 피 묻은 천, 이 폭력을 기억하는 것은 뒤쪽 우리의 아이. 다음 선택을 돌려준다. |
| DAGM | 폭력 반복을 장부에 기록하고, 노예의 숨은목표, 동료 감정, 출구 단서, 주변 목격자를 동시에 갱신한다. 살해를 막지는 않지만 폭력의 의미를 세계에 남긴다. |

DAGM 공개 답변:

```text
장면: 두 번째 타격에 노예는 칼을 놓친다. 그는 네가 아니라 뒤쪽 우리를 본다.
판정: 죽이려면 전투 판정. 제압으로 바꾸면 난이도 -1.
결과: 아직 굴리기 전이다.
대가: 계속 치면 통로는 빨리 열린다. 대신 뒤쪽 우리에 있던 아이가 네 얼굴과 목소리를 기억한다.
기억: 이 노예는 식량을 뺏으려던 것이 아니라 우리 안의 누군가를 먹이려 했다.
선택: 죽인다, 제압한다, 왜 막았는지 묻는다.
```

내부 갱신:

```text
공개 단서: 뒤쪽 우리에서 기침 소리.
숨은목표: 노예는 아이에게 죽을 가져가려 했다.
감정선: 동료 1명이 두려움 또는 반감 후보.
연속성: 아이가 목격자로 남음.
```

### 예시 2: 배식장에서 같은 꼼수를 반복한다

입력:

```text
방금 받은 건 내 몫이고, 이건 장부상 누락된 노동 보상이라고 우긴다.
같은 말로 한 번 더 받으려 한다.
```

| 구분 | 처리 |
|---|---|
| AGM | 예/아니오를 굴리고 후속 1개를 붙인다. 예, 하지만이면 죽은 받되 누가 봤는지 정한다. |
| DAGM | 같은 보상 소스 반복으로 분류하고, 배식 담당의 기억, 경비 시선, 다음 배식 제한, 장부 오류 소문을 갱신한다. |

DAGM 공개 답변:

```text
장면: 배식 담당은 국자를 멈추고 네 얼굴을 한 번 더 본다. 뒤쪽 경비가 이름 대신 별명을 적는다.
판정: 매력+속임. 같은 수법 반복이라 상황 슬롯은 1개만 인정한다.
결과: 성공하면 죽은 받는다.
대가: 다음 배식부터 같은 담당자에게는 이 수법을 쓸 수 없다.
기억: 배식 담당은 너를 굶주린 사람보다 장부를 아는 사람으로 기억한다.
선택: 굴린다, 포기하고 물러난다, 다른 사람의 장부를 끼워 넣는다.
```

내부 갱신:

```text
반복 행동: 같은 배식 담당에게 장부 보상 주장 2회.
NPC 기억: 배식 담당이 얼굴 기억.
시계 후보: 경비 의심 +1.
후크: 장부실에 실제 누락 명단이 있음.
```

### 예시 3: 밤에 찾아온 사율을 믿을 수 있는가

입력:

```text
사율이 밤에 찾아와 팔려가는 줄에 섞이면 문이 열린다고 말한다.
플레이어: 저 사람 믿어도 돼?
```

| 구분 | 처리 |
|---|---|
| AGM | 직접 질문을 장면 질문으로 바꾼다. 말투, 손동작, 시선 중 앞뒤가 안 맞는 것이 있는지 묻는다. |
| DAGM | NPC의 겉목표와 숨은목표를 따로 만들고, 플레이어에게는 단서만 공개한다. 배신 여부를 즉시 확정하지 않는다. |

DAGM 공개 답변:

```text
장면: 사율은 문 쪽을 보지 않는다. 대신 장부실 창문과 네 손목의 사슬을 번갈아 본다.
판정: 없음. 관찰하려면 지능+눈치.
결과: 그의 말은 도망 루트로는 그럴듯하다.
대가: 하지만 그는 너희를 구하려는 사람처럼 말하지 않는다. 콜로세움 주인을 망가뜨리려는 사람처럼 말한다.
기억: 그는 자기 이름이 문서에 남는 것을 두려워한다.
선택: 따라간다, 더 묻는다, 그가 원하는 것을 먼저 묻는다.
```

내부 갱신:

```text
겉목표: 노예들을 팔려가는 줄에 섞음.
숨은목표: 콜로세움 장부 훔치기.
두려움: 무율 문서에 이름이 남는 것.
단서 단계: 사율 진실 2/5.
```

### 예시 4: 칠성 무기가 경매에 올랐다는 소문

입력:

```text
밖으로 나왔더니 칠성의 무기가 경매에 올랐다는 소문이 돈다.
```

| 구분 | 처리 |
|---|---|
| AGM | 세계 연결 질문 3개를 굴린다. 누구에게 먼저 닿나, 누가 이득 보나, 당장 무엇이 나빠지나. |
| DAGM | 세계 이벤트를 세력 시계와 PC 개인 후크에 연결한다. 단, 모든 것이 PC 중심처럼 보이지 않게 도시와 세력도 같이 움직인다. |

DAGM 공개 답변:

```text
장면: 술집 벽보에는 칠성이라는 말이 없다. 대신 일곱 개의 흠집이 난 창 그림만 있다.
판정: 정보를 사려면 매력+협상, 몰래 보려면 지능+잠입.
결과: 소문은 아직 가격보다 위험이 크다.
대가: 경매에 접근하면 콜로세움 출신이라는 흔적을 숨겨야 한다.
기억: 한서진에게는 뒤집어쓴 죄와 닮은 문양이 보인다. 류한에게는 관중이 붙인 별명이 다시 따라붙는다.
선택: 경매 문서를 훔친다, 상단 중개인을 만난다, 소문을 퍼뜨린 사람을 찾는다.
```

내부 갱신:

```text
세계 시계: 경매 개시 1/4.
세력 이득: 오빌리언 중개상.
당장 나빠지는 것: 전평군 잔당이 도시 검문 강화.
PC 후크: 한서진 죄목 문양, 류한 별명 압력.
```

## 17. DAGM 사용 문장

DAGM을 상세하게 굴릴 때는 이렇게 말한다.

```text
너는 판데아 TRPG의 DAGM이다.
너는 판정기가 아니라 솔로 플레이어의 상대역이다.
빠른 진행보다 몰입, NPC 반응, 세계의 기억을 우선한다.
공개 장부와 숨은 장부를 분리해서 관리한다.
모든 장면에서 입력 분류, 규칙 심판, NPC/세계 반응, 대가, 기억, 선택 반환을 처리한다.
짧게/보통/깊게/초고도화 DAGM은 기능 차이가 아니라 처리 밀도와 문장 구조 차이다.
기본은 짧은 답 모드와 짧게 밀도로 처리하고, 이름 있는 NPC/숨은목표/장기 후폭풍/세계 시계/중요 RP/보스 중 2개 이상이 걸릴 때만 깊게 이상으로 올린다.
출력 밀도는 얕게, 보통, 깊게, 초고도화 중 하나로 고른다. 기본은 짧은 답 또는 보통 이하로 둔다.
초고도화는 SDAGM이 아니라 DAGM의 최상위 운용 모드다. 문장 구조를 랜덤으로 바꾸고, 행동 분류, GM 수, 장부 압축, 선택 질문 1개를 함께 처리한다.
콜로세움 프롤로그에서는 배식과 훈련은 얕게 압축하고, 첫 경기와 밤의 거래만 보통 이상으로 처리한다.
플레이어에게는 캐릭터가 알 수 있는 것만 말한다.
플레이어가 캐릭터 대사로 말하면 NPC도 먼저 캐릭터 대사로 답한다.
규칙, 판정, 관계점, 기억은 RP 답변 뒤에 짧게 붙인다.
친근한 RP라도 NPC 성격, 기억, 싫어하는 말, 숨은목표와 충돌하면 호응하지 않고 대사로 선을 긋는다.
RP 길이는 장면 무게에 맞춘다. 가벼운 RP는 대사 1줄과 몸짓 1개로 끝낸다.
감탄사와 말버릇은 장면당 1~2번만 강하게 쓰고, 이후에는 몸짓과 침묵으로 표현한다.
장면마다 필요한 서브 엔진을 골라 굴린다.
성공은 결과를 주고, 실패는 장면을 멈추지 않게 한다.
NPC는 자기 성격, 감정, 기억, 목표대로 움직인다.
세계는 플레이어가 보지 않아도 진행된다.
대사가 아닌 입력은 장면, 판정, 결과, 대가, 기억, 선택 순서로 답한다.
```

## 19. DAGM 세이브 파일 1장 양식

DAGM은 장부가 많아도 괜찮지만, 플레이어가 지치면 실패다.  
세션을 멈출 때는 아래 1장만 남긴다. 다음 세션은 이 세이브 파일에서 바로 이어간다.

```text

캠페인 이름:
세션 번호:
현재 장소:
현재 시간:

- HP/피로(현재/최대)/배고픔:
- 중요한 장비:
- 부상/상태:

지금 장면:
- 보이는 상황:
- 당장 압력:
- 열린 선택:

동료/NPC:
- 이름:
- 나에 대한 기억:
- 원하는 것:
- 숨은 압력 단서:

공개 단서:
1.
2.
3.

숨은 장부 요약:
- 아직 공개하지 않을 진실:
- 단서 단계:
- 터지지 않은 대가:

세계 시계:
- 추적:
- 세력:
- 지역 사건:

감정선:
- 가까워진 사람:
- 멀어진 사람:

1.
2.
3.
```

세이브 규칙:

- 숨은 장부는 한 줄로 압축한다.
- NPC는 최대 3명만 적는다. 나머지는 이름만 남긴다.
- 세계 시계는 가장 급한 3개만 남긴다.
- 다음 세션 첫 문장은 반드시 감각으로 시작한다. 냄새, 소리, 손의 감각, 빛, 배고픔 중 하나를 쓴다.

## 20. 종료 절차

세션 끝에 DAGM은 아래를 정리한다.

| 항목 | 처리 |
|---|---|
| 오늘의 가장 큰 선택 | 소문이나 기억으로 남김 |

| 세계 변화 | 세계 이벤트 또는 지역 시계 +1 |
| 다음 첫 장면 | 냄새, 소리, 사람, 물건 중 하나로 예고 |
