# TRPG 작업실

기존 판데아 TRPG 묶음은 `나의 판타지/` 폴더에 보존했습니다.

- `나의 판타지/app/`: 캐릭터 시트, 캐릭터 이미지 작업대, 주사위
- `나의 판타지/AGM/`: AGM/DAGM
- `나의 판타지/LLM/`: 아날로그 LLM
- `나의 판타지/*.md`: 룰북, 요약, 첫 플레이 키트, AGM, DAGM 문서

새 TRPG 작업:

- `저승퇴마전/`: 물리 퇴마사, 12 저승사자, 파계사자의 반란을 다루는 액션 퇴마 TRPG
- `저승퇴마전/저승퇴마전_사자록_v0.1.md`: 12사자 전용 파일
