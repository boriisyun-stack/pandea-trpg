# 판데아 TRPG 요약 v1.21

이 문서는 처음 플레이할 때 보는 빠른 규칙이다.  
자세한 표, 엔딩, 제작, 보스, 세계 이벤트는 본편 룰북을 본다. 빠른 진행은 AGM, 외로운 솔로 캠페인 감각은 DAGM을 본다.

판데아 TRPG는 솔로 플레이에 특화되어 있다.  
2인 이상 플레이도 지원하지만, 자동 질문, 동료 카드, AGM/DAGM, 세이브 시트는 혼자서도 장면과 NPC가 살아 움직이게 하려고 들어간 도구다.

처음 10분에는 이 문서보다 [첫 플레이 키트](판데아_TRPG_첫플레이키트_v1.21.md)를 먼저 본다.  
v1.21은 새 룰을 추가하지 않고, 첫 플레이 도구와 안내를 정리한 판이다.

처음 플레이의 기준 질문은 하나다.

```text
그래서 지금 무엇을 굴리나?
```

이 질문에 바로 답하지 않는 세계관, 전설, 제작, 보스, 캠페인 정보는 첫 세션에서 미뤄도 된다.

## AGM과 DAGM

AGM과 DAGM은 진행자가 없을 때 세계의 반응을 맡는 보조 진행자다.  
둘 다 플레이어 행동을 대신 정하지 않고, 숨은 진실을 단서 없이 바로 공개하지 않는다.

| 구분 | AGM | DAGM |
|---|---|---|
| 핵심 | 빠른 질문과 짧은 답 | 장면, 기억, 비밀, 세계 반응의 연속성 |
| 쓰는 때 | 막히는 순간, 가벼운 판단, 테이블 속도 유지 | 솔로 몰입, 중요한 NPC, 장기 후폭풍, 숨은 단서 |
| 기록 | 장면 끝 3줄이면 충분 | 공개/숨은/연속성 장부를 압축해서 유지 |
| 출력 | 짧은 판정 보조 | 장면 문장, NPC 반응, 대가, 기억, 다음 압력 |

처음에는 AGM으로 빠르게 넘기고, 외로운 솔로 캠페인 감각이 필요할 때 DAGM을 쓴다.  
DAGM의 짧게/보통/깊게/초고도화는 기능 차이가 아니라 출력, 문장 구조, 장부 갱신량의 차이다. 초고도화는 새 엔진이 아니라 DAGM을 GM처럼 말하게 하는 최상위 운용 모드다.

DAGM 상세 규칙은 운영 매뉴얼에 가깝다. 요약에서는 `중요 장면에 쓰는 상세 상대역`이라고만 기억한다.

## 1. 이 게임은 뭐 하는 게임인가

당신은 콜로세움의 노예로 첫 장면을 시작한다.  
처음 목표는 굶지 않고, 죽지 않고, 필수 생존 체크 8개와 출구 단서 1개를 얻은 뒤 밖으로 나가는 것이다.

하지만 진짜 이야기는 밖으로 나간 뒤 시작된다.  
판데아를 떠돌며 도감을 쓰고, 의뢰를 받고, 적에게 쫓기고, 세계가 변하는 것을 본다.

첫 플레이 권장 진행은 `콜로세움 1세션 -> 첫 바깥밤 -> 판데아 방랑 본편`이다.

## 2. 필요한 것

- 육면체 주사위 여러 개
- 캐릭터 1명 또는 2명
- 종이 또는 메모장
- 본편 룰북

진행자가 없어도 된다.  
모르면 AGM(Artificial Game Master)에게 묻는다.

## 3. 판정 방법

```text
주사위 수 = 능력치 + 기술 + 도구 슬롯 + 상황 슬롯 + 정보 슬롯
```

d6을 굴려서 5나 6이 나오면 성공 1개다.

| 필요한 성공 | 난이도 |
|---:|---|
| 1 | 쉬움 |
| 2 | 보통 |
| 3 | 어려움 |
| 4 | 치명적 |
| 5 | 영웅적 |
| 6+ | 전설적 |

성공 0개면 실패한다.  
1이 많이 나오면 꼬임이 생긴다.

실패해도 장면은 멈추지 않는다. 대가, 단서, 여파 중 하나를 남기고 다음 선택으로 간다.

첫 세션은 1~2성공 중심으로 굴리고, 3성공 이상은 중요한 단서, 위험한 전투, 탈출의 마지막 문에 쓴다.

판정 슬롯:

| 슬롯 | 들어가는 것 | 제한 |
|---|---|---|
| 도구 | 무기, 도구, 집중체, 장비 마법부여 | 가장 높은 1개 |
| 상황 | 유리, 음식, 지형, 동료 도움, 준비 행동 | 가장 높은 1개 |
| 정보 | 도감 약점, 보스 약점, 상태 이상 공략, 조사 단서 | 가장 높은 1개 |

최종 주사위는 10d를 넘지 않는다.

반복과 꼼수 바로 처리:

| 플레이어가 이렇게 하면 | 바로 적용 |
|---|---|
| 같은 보상 또 노림 | 보상 소스 확인 |
| 같은 장면 같은 목적 반복 | 키워드 소진 |
| 계속 쉼/기다림 | 정체/위험 회피 카드 |
| 사람 계속 때림 | 충동형 파괴 카드 |
| 메타로 정답 질문 | 감각 질문으로 변환 |
| 장면을 한 방에 삭제 | 작은 효과로 변환 또는 강함의 대가 |

## 3-1. 콜로세움에서 꼭 써볼 시스템

첫 세션에서는 아래 `필수 생존 체크 8개`만 끝내면 밖으로 나간다.

| 체크 | 시스템 |
|---|---|
| [ ] | 판정 슬롯 |
| [ ] | 유리/불리, 기회, 꼬임 |
| [ ] | 배고픔, 음식, 피로 |
| [ ] | NPC 성향과 대화 |
| [ ] | AGM 질문 |
| [ ] | 거리, 공격, 방어 |
| [ ] | 적 AI와 현재 압력 |
| [ ] | 방호, 내구, 부상 또는 상태 이상 |

심화 체크 5개는 자연스럽게 나오면 하고, 아니면 첫 바깥밤이나 첫 의뢰로 넘긴다.

| 체크 | 시스템 |
|---|---|
| [ ] | 도감 판정과 정보 슬롯 |
| [ ] | 수리, 제작, 개조 중 하나 |
| [ ] | 마법, 율법, 화신 중 하나를 자연스럽게 사용하거나 대응 |
| [ ] | 보상 소스와 반복 키워드 소진 |
| [ ] | 추적 위험, 자유평판, 세계반응 시계 중 하나 |

필수 생존 체크 8개와 출구 단서 1개가 있으면 본편 룰북 21장의 출구 개방을 굴리고 23장으로 간다.

체크를 한 장면씩 따로 넣지 않는다.  
배식, 훈련, 첫 경기, 밤의 거래, 출구 개방에 2~3개씩 묶고, 남는 것은 밖으로 넘긴다.

| 장면 | 자연스럽게 체크할 것 |
|---|---|
| 배식 | 판정 슬롯, 배고픔/음식, NPC 성향 |
| 훈련 | 유리/불리, 기회/꼬임, 보상 소스 |
| 첫 경기 | 거리, 공격/방어, 적 AI, 방호/내구/부상, 도감 |
| 밤의 거래 | AGM 질문, NPC 관계, 수리/제작/개조 중 어울리는 것 |
| 출구 개방 | 출구 단서, 추적 위험 또는 자유평판 |

전투 중에는 매 라운드 `거리 -> 압력 -> 적 AI -> 공격/방어 -> 피해/방호 -> 내구/상태 -> 여파`만 적는다.

마법/율법/화신 PC가 없다면 억지로 넣지 않는다. 방문자, 경비, 귀족석 소동으로 자연스럽게 보일 때만 체크한다.

| 첫 경기 | 2~4라운드만 싸운다 | 거리, 공격/방어, 적 AI, 방호/상태 |
| 경기 후 | 상대, 음식, 장비 중 하나를 관찰한다 | 도감 또는 이월 |
| 밤의 거래 | 방문자와 단서 하나를 얻는다 | AGM 질문, 관계, 수리/마법은 선택 |
| 출구 | 단서로 문이 열린다 | 추적 위험 또는 자유평판 |

## 4. 캐릭터 만들기

순서대로 굴린다.

1. 노예가 된 이유 2d6
2. 종족 1d6
3. 특수 운명과 노예 낙인 1d6
4. 성별 1d6
5. 모습 여러 표
6. 능력치 7개 각각 1d6
7. 기술 5개 선택
8. 시작 장비 1d6
9. 직업 씨앗 선택
10. 목표와 두려움 작성

중요:

- 노예가 된 이유는 사회적 사건이다.
- 종족은 몸과 혈통이다.
- 특수 운명은 장부 기록, 소문, 낙인이다.
- 이 셋을 섞지 않는다.
- 능력치는 먼저 굴리고, 앞 단계 보너스를 더한다.
- 시작 최종 능력치는 4를 넘을 수 없다.

능력치는 7개다.

| 능력치 | 쓰임 |
|---|---|
| 근력 | 때리기, 들기, 버티기 |
| 민첩 | 피하기, 빠르게 움직이기 |
| 체력 | HP, 독, 배고픔 |
| 지성 | 도감, 함정, 전략 |
| 정신 | 공포, 고문, 율법 |
| 마나 | 그라듀 캐스팅, 마나 감지 |
| 매력 | 말솜씨, 외모, 설득, 공연, 후원자 |

기술은 주사위로 정하지 않는다.  
플레이어가 직접 5개 고른다. 처음 고른 기술은 전부 `초급`이다.

출신/노예 이유의 기술 +1이 시작 기술과 겹치면 중급이 되지 않고 `숙련표식 1개`로 바꾼다.  
시작 시점에는 어떤 기술도 중급 이상이 될 수 없다.

기술은 성장한다.

```text
초급 -> 중급 -> 상급 -> 고급 -> 장인 -> 명인 -> 전설
```

예:

- 초급 검술
- 중급 검술
- 상급 검술
- 초급 마법학
- 중급 생존

성별은 남자, 여자, 중성적인 생김새만 쓴다.  
중성적인 생김새는 성별을 남자/여자 중에서 고르고, 외형만 중성적으로 보이는 것이다.

모습은 눈색, 귀 모양, 머리/피부, 매력 인상, 관중 별명으로 정한다.  
관중 별명은 `000한 00` 형식이다.

직업 씨앗은 밖으로 나간 뒤 어떤 삶을 살지 정한다.

| 직업 씨앗 | 본편 방향 |
|---|---|
| 검투사 | 결투, 호위, 챔피언 |
| 도감기록자 | 마수 조사, 기록 판매 |
| 마수사냥꾼 | 추적, 해체, 재료 |
| 마법사 | 마법 의뢰, 실험 |
| 장인 | 제작, 수리, 유물 복원 |
| 협상가 | 상단, 후원자, 도시 정치 |

처음이면 35장 샘플 캐릭터에서 `미라 + 보리` 조합을 추천한다. 미라는 도감과 생존을 맡고, 보리는 조련과 관계 장면을 맡아서 전투 없이도 장면이 잘 굴러간다.

## 5. 자주 쓰는 판정

| 상황 | 판정 |
|---|---|
| 공격 | 근력/민첩 + 해당 무기 기술 |
| 방어 | 민첩+회피, 근력+해당 무기 기술/방패술, 체력+방호 중 선택. 버티기 방호는 최대 +2d |
| 도감 | 지성 + 도감 |
| 마법 | 마나 + 마법학 |
| 율법 | 정신 + 율법 |
| 설득 | 매력 + 설득 |
| 관중 도발 | 매력/정신 + 공연. 한 판정 호응 +2까지 |
| 생존 | 체력/지성 + 생존 |
| 제작 | 지성/근력 + 제작 |
| 직업 행동 | 직업에 맞는 능력 + 관련 기술 |

## 6. 하루 흐름

콜로세움 안:

1. 새벽 배식
2. 오전 훈련
3. 오후 경기
4. 밤 우리

콜로세움 밖:

1. 길 위의 아침
2. 낮 이동 또는 의뢰
3. 해질녘 사건
4. 밤 야영 또는 도시 체류

도시 체류:

```text
검문 -> 숙소 -> 생활 행동 1개 -> 비용/빚 정산 -> 밤 소문
```

| 생활 행동 | 판정 | 성공 |
|---|---|---|
| 일거리 찾기 | 매력/지성 + 설득/직업 기술 | 식량, 체류비, 작은 의뢰 중 하나 |
| 목욕과 옷 정리 | 매력/체력 + 생존 | 낙인 숨기기나 첫 반응 +1 |
| 장비 수리 | 지성/근력 + 제작 | 내구 +1 또는 노후화 제거 |
| 음식 사기/조리 | 매력/지성 + 생존/설득 | 배고픔 -2 또는 다음 생존 +1d |
| 빚 갚기/미루기 | 매력/정신 + 설득 | 빚 제거 또는 추적 위험 증가 무시 |
| 소문 듣기 | 지성/매력 + 감각/설득 | NPC, 의뢰, 가격, 검문, 위험 단서 |

계절과 날씨는 음식, 이동, 장비를 움직인다. 도시나 지역이 밋밋하면 계절 하나와 날씨 d6을 붙인다.

NPC가 다시 나오면 세 줄만 적는다.

```text
전에 이 NPC와 있었던 일: ______
지금 달라진 입장: ______
플레이어에게 원하는 것: ______
```

## 7. AGM

진행자가 없거나 모르는 것이 있으면 AGM(Artificial Game Master)에게 묻는다. 빠른 진행이 필요하면 [AGM](판데아_TRPG_AGM_v1.21.md), 외로운 솔로 캠페인에서 상대역이 필요하면 [DAGM](판데아_TRPG_DAGM_v1.21.md)을 쓴다.  
질문은 캐릭터가 알 수 있는 방식으로만 묻는다.

| 상황 | 추천 밀도 |
|---|---|
| 배식, 이동, 짧은 조사, 가벼운 대화 | AGM 또는 DAGM 짧은 답 모드 |
| NPC 비밀, 밤의 거래, 첫 경기, 장기 후폭풍 | 보통 밀도 DAGM |
| 보스, 배신, 도시 사건, 엔딩 | 전체 밀도 DAGM |

AGM은 장면 끝에 `원하는 것`, `기억/오해`, `다음에 움직일 것` 3줄만 적어도 깊이가 유지된다.  
DAGM은 모든 장면에 같은 밀도로 답하지 않는다. 기본은 짧은 답 모드다. 기능을 빼는 것이 아니라, 출력과 장부 갱신을 압축한다.
DAGM은 판정기가 아니라 상대역이다. 플레이어가 RP로 말하면 먼저 NPC 대사로 답하고, 숫자는 뒤에 짧게 붙인다.

DAGM 표의 단어는 그대로 읽지 않는다. `감각 -> 이상한 물건/사람 -> 압력을 보여주는 행동 -> 건드릴 수 있는 틈 -> 무엇을 하나?` 순서로 장면 문장으로 바꾼다. 선택지 목록은 초심자나 막힌 장면에서만 써도 된다.

좋은 질문:

- 경비 발소리가 느슨한가?
- 이 피 냄새가 최근 것인가?
- 상인이 내 별명을 알아들은 눈치인가?

나쁜 질문:

- 이 NPC가 배신자인가?
- 여기 보스 약점이 있나?
- 정답 선택지가 뭔가?

나쁜 질문은 이렇게 바꾼다.

| 직접 질문 | 바꿔 묻기 |
|---|---|
| 이 NPC가 배신자인가? | 말투, 시선, 손동작 중 앞뒤가 안 맞는 게 있나? |
| 보스 약점이 뭐야? | 보스가 감추거나 피하는 부위, 재료, 소리가 있나? |
| 이 선택지가 정답이야? | 이 선택을 하면 가장 먼저 잃는 게 뭔가? |
| 여기에 함정 있어? | 바닥, 먼지, 냄새, 소리 중 이상한 게 있나? |

질문이 막히면 아래 중 하나로 시작한다.

- 무엇이 먼저 보이나?
- 무엇이 너무 조용하거나 깨끗한가?
- 누가 가장 이득을 보나?
- 시간이 지나면 무엇이 나빠지나?
- 내가 쓰려는 방법의 대가는 무엇인가?

그래도 질문이 안 떠오르면 장면 유형만 고른다.

| 장면 유형 | 자동 질문 |
|---|---|
| 반복 보상/꼼수 | 같은 소스에서 다시 이득을 얻는가? |
| 장치/지형 악용 | 자동 성공인가, 상황 이점인가? |
| 충동형 파괴/폭력 | 부서진 것은 무엇이고, 누가 기억하나? |
| 정체/위험 회피 | 멈춰 있는 동안 무엇이 움직이나? |
| NPC 의도 | 이 NPC는 지금 어떤 사람으로 움직이나? |
| NPC 신뢰 | 겉목표와 숨은 압력이 어긋나나? |
| 전투 | 다음 라운드에 바뀌는 압력은 무엇인가? |
| 세계 이벤트 | 이 사건은 PC 중 누구에게 먼저 닿나? |

자동진행을 쓰면 결과가 무엇이든 후속 질문 1개를 반드시 묻는다.

NPC를 만나면 예/아니오를 묻기 전에 의도 AGM을 굴린다. 4d6으로 `성격 -> 감정 -> 플레이어에 대한 기억 -> 겉의도`를 만든다.

| d6 | 성격 | 감정 | 기억 | 겉의도 |
|---:|---|---|---|---|
| 1 | 보호자 | 겁먹음 | 낯선 위협 | 도움을 원함 |
| 2 | 계산가 | 초조함 | 번호/별명 앎 | 뭔가 숨김 |
| 3 | 생존자 | 굶주림 | 닮은 사람 떠올림 | 거래 원함 |
| 4 | 이름욕심 | 분노 | 원한과 겹쳐 봄 | 시험함 |
| 5 | 신념형 | 죄책감 | 팔 정보로 봄 | 도망치려 함 |
| 6 | 복수자 | 침착함 | 같은 처지로 봄 | 적의 |

중요 NPC는 [AGM 별도 파일](판데아_TRPG_AGM_v1.21.md#8-npc-의도-agm)에서 숨은 압력도 굴린다.

압력이 있는데 두 번 연속 `쉰다`, `기다린다`, `아무것도 안 한다`, `위험한 건 피한다`가 나오면 정체/위험 회피 카드를 쓴다.

```text
멈춰 있는 동안 움직인 것은 ______다.
사라지는 기회는 ______다.
이제 선택을 요구하는 것은 ______다.
```

d6: 1 시계 +1, 2 배고픔/피로/식량 압박, 3 누군가 먼저 움직임, 4 안전한 길 폐쇄, 5 보상 감소, 6 작은 사람이 도움 요청. 세 번째 정체부터는 장면 압력도 1개 더 갱신한다.

가능성을 정하고 2d6을 굴린다.

| 결과 | AGM 답 |
|---:|---|
| 3 이하 | 아니오. 그리고 GM 강한 수 |
| 4~5 | 아니오. 하지만 작은 흔적 |
| 6~8 | 예. 하지만 대가, 지연, 오해 |
| 9~10 | 예 |
| 11 이상 | 예. 그리고 기회나 단서 |

`예, 하지만`이 나오면 아래에서 하나를 고른다.

| d6 | 하지만 |
|---:|---|
| 1 | 시간이 든다. 시계 하나 +1 |
| 2 | 소리가 나서 누군가 반응한다 |
| 3 | 식량, 돈, MP, 피로, 내구 중 하나를 1 잃는다 |
| 4 | NPC 하나가 오해하고 성향대로 움직인다 |
| 5 | 일부만 된다. 범위, 품질, 지속시간 중 하나 감소 |
| 6 | 더 큰 문제의 이름, 냄새, 표식이 드러난다 |

`예, 하지만`을 일반 AGM으로 처리했다면 그 다음 아래 후속 질문 중 하나를 묻는다.  
자동진행을 쓰는 중이면 이미 후속 질문을 묻는 것이 규칙이므로 한 번만 묻는다.

| d6 | 후속 질문 |
|---:|---|
| 1 | 누가 이걸 봤나? |
| 2 | 누가 이걸 잘못 이해했나? |
| 3 | 어떤 흔적이 남았나? |
| 4 | 무엇이 다음 장면까지 따라오나? |
| 5 | 언제 이 대가가 터지나? |
| 6 | 이 일로 누가 이득을 보나? |

AGM은 항상 네 가지로 답한다.

1. 보이는 것
2. 느껴지는 것
3. 변하는 것
4. 지금 선택할 수 있는 것

중요한 장면이면 두 가지를 더 붙인다.

5. 숨은 압력: 시간이 지나면 무엇이 나빠지는가
6. 후크: 다음 장면으로 이어지는 냄새, 소리, 물건, 이름

장면이 밋밋하면 d6 세 번으로 AGM 패널을 만든다.

```text
첫 d6: 장면 의도
둘째 d6: 숨은 압력
셋째 d6: 드러나는 단서
```

## 8. 혼자서 하는 법

혼자 할 때는 동료 카드 2장을 만든다.

동료는 직접 주인공처럼 행동하지 않는다.  
대신 의견을 내고, 장면을 흔들고, 보조 +1d를 준다.

중요한 선택 앞에서는 이렇게 진행한다.

```text
주인공 생각 -> 동료 A 의견 -> 동료 B 의견 -> AGM 압박 -> 선택
```

세션당 한 번씩 세 가상 좌석도 쓸 수 있다.

| 좌석 | 하는 일 |
|---|---|
| 생존가 | 대가와 자원을 확인 |
| 도감가 | 단서 질문 추가 |
| 문제아 | 위험하지만 재밌는 선택지 생성 |

## 9. NPC

NPC는 직업, 성향, 세부 성향, 비밀, 목표, 성격, 말버릇을 가진다.

성향이 중요하다.

| 성향 | 움직임 |
|---|---|
| 겁먹음 | 위험하면 숨거나 도망 |
| 이해타산 | 손익을 먼저 봄 |
| 보호적 | 누군가를 지키려 함 |
| 야심적 | 이름, 돈, 권력을 원함 |
| 원한 | 복수를 원함 |
| 신념형 | 신념을 쉽게 버리지 않음 |

## 10. 몬스터

몬스터는 순간 성향보다 `고정 성향`이 중요하다.  
고정 성향은 그 괴물이 세상을 대하는 기본 방식이다.

v1.21부터는 `종별 성격`도 본다.  
고정 성향은 본능이고, 종별 성격은 그 종이 장면에서 풍기는 개성이다.

예:

- 기본 마수: 군집 포식
- 리더 마수: 지휘 본능
- 워커: 약탈 지능
- 띵커 워커: 위장 지능
- 패니커: 공황 방사
- 스키니호스: 장거리 광기
- 모래아귀: 유인 포식
- 파이로: 화염 본능
- 사이렌: 음파 해체
- 무빙아머: 기사 잔향

종별 성격 예:

- 트램비: 겁 많지만 기억력이 좋음
- 주닭: 자존심 강한 소형 싸움꾼
- 워커: 탐욕스러운 약탈자
- 띵커 워커: 어설픈 흉내쟁이
- 스키니호스: 멀리서 미치게 하는 관찰자
- 무빙마스터: 예의 바른 살인 교관

몬스터 행동은 이렇게 본다.

```text
고정 성향 -> 종별 성격 -> 현재 압력 -> 행동표
```

## 11. 전투 핵심

거리는 6단계다.

| 단계 | 거리 |
|---:|---|
| 0 | 붙음 |
| 1 | 짧음 |
| 2 | 가까움 |
| 3 | 중거리 |
| 4 | 멀리 |
| 5 | 시야끝 |

공격이 방어보다 성공 수가 많으면 맞는다.

```text
피해 = 무기 피해 + 순성공 수 - 방호
```

방호는 피해 감소에는 전부 쓴다.  
하지만 `체력 + 방호`로 버티기 방어를 할 때는 방호가 아무리 높아도 최대 +2d까지만 더한다.

HP가 0이면 쓰러짐 표를 굴린다.

상태 이상은 출혈, 화상, 중독, 공포, 속박, 혼란, 침묵, 취약, 넘어짐, 낙인이 있다.

## 12. 그라듀 마법

마법은 주문을 외우는 것이 아니라 그라듀를 조합한다.

기본 그라듀 예시:

- 운동에너지: 전방
- 운동에너지: 후방
- 운동에너지: 상향
- 점화
- 가스 생성
- 물방울
- 빛
- 소리
- 마나실
- 감지
- 생체자극

마법학이 성장하면 한 번에 더 많이 조합한다.

| 단계 | 조합 가능 |
|---|---:|
| 초급 | 1개 |
| 상급 | 2개 |
| 장인 | 3개 |

## 13. 도감

괴물, 음식, 장비, 세력, 전설을 보면 도감 판정을 할 수 있다.

```text
지성 + 도감
```

마수 도감은 5칸짜리 빠른 도감이다.

- 관찰
- 약점
- 식용
- 포획
- 재료

칸을 채울수록 보너스가 생긴다.  
완성하면 제작, 전투, 소문 판매에 도움이 된다.

괴물, 음식, 유물, 지역 사건까지 길게 기록할 때는 룰북 18-2장의 6칸 전체 도감 카드로 확장한다.

## 14. 세계는 계속 움직인다

세션 끝에 세계 이벤트 d66을 굴린다.  
이 사건은 플레이어와 직접 관련이 없어도 일어난다.

예:

- 전평군 잔당이 도시를 점령
- 새 마수 발견
- 루멘티스 성전 선포
- 칠성 목격담
- 대가뭄
- 노예 반란

## 15. 메타게이밍 금지

플레이어가 아는 것과 캐릭터가 아는 것은 다르다.

- 도감에 없는 약점은 보너스를 못 받는다.
- NPC 비밀은 직접 묻지 말고 행동과 흔적을 본다.
- 보스 페이즈 조건은 처음부터 알 수 없다.
- 이전 캠페인 지식은 이번 캐릭터 지식이 아니다.

핵심 질문:

> 지금 캐릭터가 이것을 알 이유가 장면 안에 있었나?

없으면 보너스는 없다.

## 16. 슬롯과 소스

숫자는 슬롯에 들어갈 때만 올라간다.  
슬롯에 못 들어간 보너스는 주사위가 아니라 장면 효과가 된다.

| 소스 | 보상 주기 |
|---|---|
| 같은 적의 같은 약점 | 장면 1번 |
| 같은 음식 | 하루 1번 |
| 같은 동료 도움 | 장면 1번 |
| 같은 NPC | 하루 1번 |
| 같은 직업 행동 | 도시당 하루 1번 |
| 같은 장비 이름 | 세션 1번 |
| 같은 도감 선언 | 세션 1번 |
| 같은 제작 재료 | 장비 1개에 1번 |

같은 장면에서 같은 행동 키워드가 두 번 성공하면 그 키워드는 소진된다.  
소진된 키워드로는 보상, 약점, 자유점수, 재료, 돈, 호응을 더 얻지 못한다.

## 16-1. 강함의 대가

너무 빠르게 강해지면 강함의 대가를 굴린다.  
d6 두 개로 1~12를 만든다.

```text
첫 d6이 1~3이면 +0
첫 d6이 4~6이면 +6
둘째 d6 값을 더한다
```

결과는 추적, 빚, 동료 갈등, 장비 손상, 세계반응, 보스급 관심 중 하나로 나타난다.  
자세한 12개 표는 본편 룰북 1장의 강함의 반동을 본다.

그래도 버티면 단계가 오른다.

| 단계 | 언제 쓰나 |
|---|---|
| 강함의 대가 | 개인과 주변이 반응 |
| 최강의 대가 | 세력과 운명이 반응 |
| 신화의 대가 | 세계 법칙과 신화가 반응 |

최강의 대가와 신화의 대가는 HP, 방호, 내성으로 줄이지 못한다.  
피해가 아니라 세계가 캐릭터를 사건으로 취급하기 시작한 것이다.

## 17. 엔딩

엔딩은 적을 죽이는 것이 아니라, 캐릭터가 자기 질문에 답하는 장면이다.

최종 보스는 세 층으로 만든다.

| 층 | 의미 |
|---|---|
| 개인 | 나에게 상처 준 존재 |
| 세계 | 판데아의 큰 문제 |
| 내면 | 내가 어떤 사람이 될지 |

최종 보스 양식:

```text
내 개인의 적:
판데아의 세계 문제:
내면의 질문:

1페이즈 얼굴:
- 드러나는 원한:
- 필요한 단서/기억:

2페이즈 세계:
- 전장이 어떻게 커지는가:
- 희생되거나 위험해지는 것:

3페이즈 이름:
- 보스가 제안하는 이름:
- 내가 선택할 이름:

승리 조건:
- 피해:
- 약점 단서:
- NPC 기억:
- 도감/흉터/생활 기억:
```

엔딩 뒤에는 이것을 말한다.

1. 마지막으로 먹은 음식
2. 고친 장비 또는 버린 장비
3. 나를 기억하는 NPC
4. 내가 남긴 도감 한 줄
5. 나 때문에 바뀐 세계
6. 그래도 바뀌지 않은 문제

마지막 문장은 캐릭터의 이름으로 끝낸다.

## 18. v1.21 실전 데이터는 이렇게 쓴다

본편으로 나간 뒤 장면이 비면 아래 순서로 하나씩 꺼낸다.

1. 지역을 고른다. 본편 31장 지역 카드에서 사건을 하나 굴린다.
2. 마수나 NPC가 나오면 본편 20장 적 AI나 29장 NPC 생성표로 성향과 행동 압력을 정한다.
3. 중요한 적이면 본편 32장 보스 카드 중 하나를 가져온다.
4. 조사나 보상은 본편 33장 괴물, 음식, 유물 표에서 만든다.
5. 긴 이야기가 필요하면 본편 34장 캠페인 3개 중 하나를 시작한다.

초심자는 `도감기록자의 첫 자유`가 가장 쉽다.  
사회전과 추격을 많이 하고 싶으면 `검은 장부와 해방자들`, 고레벨 판타지와 최종 보스를 원하면 `로운실의 심장과 신화의 대가`를 고른다.

## 플레이 중 1장 요약


### 기본 판정

```text
주사위 수 = 능력치 + 기술 + 도구 슬롯 + 상황 슬롯 + 정보 슬롯
```

5~6은 성공. 6이 2개 이상이면 기회 1개.  
성공 0개는 실패. 1이 절반 이상이면 꼬임.

| 성공 | 난이도 |
|---:|---|
| 1 | 쉬움 |
| 2 | 보통 |
| 3 | 어려움 |
| 4 | 치명적 |
| 5 | 영웅적 |
| 6+ | 전설적 |

### 슬롯

| 슬롯 | 예시 | 제한 |
|---|---|---|
| 도구 | 무기, 도구, 집중체 | 가장 높은 1개 |
| 상황 | 유리한 지형, 준비, 도움 | 가장 높은 1개 |
| 정보 | 도감 약점, 보스 단서 | 가장 높은 1개 |

최종 주사위는 보통 10d까지.

### 전투

거리:

| 단계 | 거리 |
|---:|---|
| 0 | 붙음 |
| 1 | 짧음 |
| 2 | 가까움 |
| 3 | 중거리 |
| 4 | 멀리 |
| 5 | 시야끝 |

선제권:

```text
민첩 + 감각
```

공격:

```text
근력/민첩 + 무기 기술 + 무기 보너스
```

방어:

| 선택 | 판정 |
|---|---|
| 회피 | 민첩 + 회피 |
| 막기 | 근력 + 무기 기술 또는 방패술 |
| 버티기 | 체력 + 방호, 방호는 최대 +2d |

피해:

```text
피해 = 무기 피해 + 순성공 수 - 방호
```

방호는 피해 감소에 전부 쓴다.  
버티기 방어 주사위에 더할 때만 최대 +2d다.

AGM 전투 순서:

```text
적 AI -> 플레이어 선언 -> 판정 -> 기회/꼬임 -> 장면 압력 1개 갱신
```

장치/지형 꼼수는 자동 승리 대신 이동 제한, 다음 공격 +1, 적 방어 -1d, 상태 이상 1단계 중 하나로 바꾼다.

사람에게 `친다`를 반복하면 먼저 충동형 파괴 카드를 쓴다.

```text
부서진 것은 ______다.
손에 남은 것은 ______다.
이 폭력을 기억하는 것은 ______다.
```

그리고 d6: 1 큰 소리, 2 잔해, 3 흔적, 4 안쪽 단서, 5 위험한 길, 6 다친 사람. 말하는 상대에게 두 번째 같은 폭력을 쓰면 이 카드를 먼저 적용하고, 세 번째부터는 장면 압력도 1개 더 갱신한다.

사람에게 반복 폭력을 쓰는 장면은 룰북 8장의 [사람 대상 반복 폭력](판데아_TRPG_룰북_v1.21.md#사람-대상-반복-폭력) 질문표도 같이 본다.

### 상태 이상

| 상태 | 핵심 효과 |
|---|---|
| 출혈 | 장면 끝 HP -1 |
| 화상 | 행동하면 피해 또는 피로 압박 |
| 중독 | 체력 판정 실패 시 악화 |
| 공포 | 정신 판정 실패 시 행동 제한 |
| 속박 | 이동 불가 |
| 혼란 | AGM/진행자가 행동 꼬임 제시 |
| 침묵 | 마법/명령 방해 |
| 취약 | 다음 피해 증가 |
| 넘어짐 | 붙음 거리 공격에 취약 |
| 낙인 | 추적/소문에 반응 |

### 마수 행동

```text
고정 성향 -> 종별 성격 -> 현재 압력 -> 행동표
```

현재 압력 d6:

| d6 | 압력 |
|---:|---|
| 1 | 굶주림 |
| 2 | 영역 수호 |
| 3 | 상처 입음 |
| 4 | 무리 우세 |
| 5 | 추적 중 |
| 6 | 조종/오염 |

### 방랑 루프

1. 지역을 고른다.
2. 의뢰나 문제를 받는다.
3. 길 위 이벤트를 굴린다.
4. NPC 또는 마수 행동표를 굴린다.
5. 전투, 협상, 도감, 생존 중 하나로 해결한다.
6. 보상과 대가를 기록한다.
7. 세션 끝 세계 이벤트를 굴린다.

### 자주 쓰는 판정

| 상황 | 판정 |
|---|---|
| 공격 | 근력/민첩 + 무기 기술 |
| 회피 | 민첩 + 회피 |
| 막기 | 근력 + 무기 기술/방패술 |
| 버티기 | 체력 + 방호, 방호 최대 +2d |
| 도감 | 지성 + 도감 |
| 마법 | 마나 + 마법학 |
| 율법 | 정신 + 율법 |
| 설득/흥정 | 매력 + 설득 |
| 배고픔 버티기 | 체력 + 생존 |
| 제작/수리 | 지성/근력 + 제작 |
| 보스 약점 찾기 | 도감/감각/마법학/율법/제작 |

### AGM 질문

비밀을 직접 묻지 말고 장면에서 확인할 수 있는 흔적을 묻는다.

| 직접 질문 | 바꿔 묻기 |
|---|---|
| 이 NPC 배신자야? | 말투, 시선, 손동작 중 앞뒤가 안 맞는 게 있나? |
| 약점이 뭐야? | 적이 감추거나 피하는 부위, 재료, 소리가 있나? |
| 함정 있어? | 바닥, 먼지, 냄새, 소리 중 이상한 게 있나? |
| 정답이 뭐야? | 이 선택의 대가는 무엇인가? |

`예, 하지만` d6:

| d6 | 하지만 |
|---:|---|
| 1 | 시간이 든다 |
| 2 | 소리가 난다 |
| 3 | 자원 하나를 쓴다 |
| 4 | NPC가 오해한다 |
| 5 | 일부만 된다 |
| 6 | 더 큰 문제의 단서가 보인다 |

자동진행을 쓰면 결과가 무엇이든 후속 하나를 굴린다.  
일반 질문에서 `예, 하지만`이 나와도 아래 후속을 하나 굴린다.

| d6 | 후속 |
|---:|---|
| 1 | 누가 봤나? |
| 2 | 누가 오해했나? |
| 3 | 무엇이 남았나? |
| 4 | 무엇이 따라오나? |
| 5 | 언제 터지나? |
| 6 | 누가 이득 보나? |

### 세션 끝

1. 배고픔, 피로, HP를 정리한다.
2. 도감 한 줄을 쓴다.
3. NPC가 기억하는 일을 하나 정한다.
4. 세계 이벤트 d66을 굴린다.
5. 다음 세션의 첫 장면을 한 문장으로 정한다.
