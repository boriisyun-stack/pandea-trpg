# 판데아 TRPG 첫 플레이 키트 v1.21

이 문서는 첫 플레이용 도구만 모은다.  
새 룰은 없다. 룰북 v1.21에 있는 규칙을 처음 1세션에서 덜 헤매게 접은 것이다.

판데아 TRPG는 솔로 플레이를 기준으로 설계된 부분이 많다.  
둘이서 해도 좋지만, 이 키트의 체크 카드, DAGM 세이브, 긴 로그 예시는 혼자서도 누군가와 TRPG를 하는 느낌을 내기 위한 도구다.

## 0. 처음 10분

처음이면 룰북을 전부 읽지 않는다.  
아래 순서만 따른다.

1. 캐릭터 1~2명을 고른다. 처음이면 `미라 + 보리`를 추천한다.
2. 첫 장면은 `콜로세움의 노예`다.
3. 목표는 `필수 생존 체크 8개 + 출구 단서 1개`를 얻고 밖으로 나가는 것이다.
4. 심화 체크 5개는 자연스럽지 않으면 첫 바깥밤으로 넘긴다.
5. 빠른 진행은 AGM, 외로운 솔로 캠페인 감각은 DAGM을 쓴다.

이 키트의 목적은 룰북을 설명하는 것이 아니라 첫 세션을 굴리는 것이다.  
플레이 중에는 계속 아래 질문으로 돌아온다.

```text
그래서 지금 무엇을 굴리나?
```

이 질문에 바로 답하지 않는 고유명사, 전설, 제작, 보스, 지역 정보는 첫 바깥밤 이후로 미룬다.

판데아의 판정은 일부러 조금 매운 편이다. 실패해도 장면은 멈추지 않고, 대가, 단서, 여파 중 하나를 남기며 앞으로 간다.

피해, 방호, 피로, 배고픔, 도감 보너스는 첫 플레이 뒤 맛을 확인하는 수치다. 첫 세션 중에는 고치지 말고 그대로 굴린다.

### AGM과 DAGM 1분 이해

AGM과 DAGM은 진행자가 없을 때 쓰는 보조 진행자다. 둘 다 플레이어의 행동을 대신 정하지 않는다. 둘 다 실패를 장면 정지로 만들지 않고, 결과와 여파를 다음 선택으로 넘긴다.

| 도구 | 한 줄 설명 | 처음에는 |
|---|---|---|
| AGM | 빠른 장면 질문기. 모르는 것을 캐릭터가 알 수 있는 방식으로 묻고 짧게 답한다 | 막히는 순간에만 쓴다 |
| DAGM | 상세 상대역. 장면, NPC 기억, 숨은 단서, 세계 반응을 장부로 이어간다 | 기본은 짧은 답 모드. 첫 경기/밤의 거래만 보통 밀도 |

AGM은 빠르고 가볍다. DAGM은 느리지만 혼자서도 누군가와 플레이하는 감각이 강하다.  
처음에는 DAGM을 전부 열지 말고, 장면 문장과 기억 1줄만 남겨도 된다.
초고도화 DAGM은 첫 세션의 기본 모드가 아니다. 이름 있는 NPC, 배신 단서, 보스, 엔딩처럼 정말 GM 상대역 감각이 필요한 장면에서만 쓴다.

## 1. 용어 입구

첫 세션에 알아야 하는 말:

| 용어 | 지금 뜻 |
|---|---|
| 판데아 | 밖으로 나간 뒤 떠돌 세계 |
| 콜로세움 | 첫 장면이자 튜토리얼 감옥 |
| 출구 단서 | 밖으로 나갈 명분, 열쇠, 사람, 소문 |
| 배고픔/피로 | 배고픔은 쌓이는 압박, 피로는 줄어드는 남은 기력 |
| 방호/내구 | 맞았을 때 줄이는 값, 장비가 버티는 힘 |
| 도감 | 괴물, 음식, 물건을 기록해 정보 이점을 얻는 기록 |
| AGM | 빠르게 장면을 굴리는 테이블용 진행 보조 |
| DAGM | 느리지만 몰입이 강한 솔로용 상대역 |

지금 몰라도 되는 전설:

| 용어 | 첫 세션에서는 |
|---|---|
| 칠성 | 아주 강한 전설급 인물들이 있다는 정도만 알면 된다 |
| 화신 | 신화적 힘의 사용자. PC가 관련 없으면 나중에 본다 |
| 무율/사율 | 율법 계열 인물과 질서. 밤 방문자가 나오면 이름만 기억한다 |
| 로운실 | 중요한 재료/신화 자원. 제작 캠페인 때 본다 |
| 코도 반 | 콜로세움 출신이라는 소문이 있는 암살왕. 지금은 소문이다 |
| 루멘티스/제도니아/전평군 | 밖의 세력 이름. 첫 바깥밤 이후 천천히 만난다 |

## 2. 콜로세움 체크 카드

기본 목표는 13개 시스템을 모두 보여주는 것이다.  
첫 플레이에서는 자연스럽지 않으면 필수 생존 체크 8개만 끝내고 출구를 연다.

필수 생존 체크 8개:

| 체크 | 시스템 | 자연 장면 |
|---|---|---|
| [ ] | 판정 슬롯 | 배식, 훈련, 첫 경기 |
| [ ] | 유리/불리, 기회, 꼬임 | 아무 판정 1회 |
| [ ] | 배고픔, 음식, 피로 | 첫 배식과 경기 후 |
| [ ] | NPC 성향과 대화 | 배식 노예나 밤 방문자 |
| [ ] | AGM 질문 | 밤의 거래나 탈출 준비 |
| [ ] | 거리, 공격, 방어 | 첫 경기 |
| [ ] | 적 AI와 현재 압력 | 첫 경기 각 라운드 |
| [ ] | 방호, 내구, 부상 또는 상태 이상 | 맞거나 막을 때 |

심화 체크 5개:

| 체크 | 시스템 | 자연스럽지 않으면 |
|---|---|---|
| [ ] | 도감 판정과 정보 슬롯 | 첫 바깥밤의 발자국, 음식, 괴물 흔적 |
| [ ] | 수리, 제작, 개조 중 하나 | 첫 야영 |
| [ ] | 마법, 율법, 화신 중 하나 | 첫 의뢰에서 목격하고 대응 |
| [ ] | 보상 소스와 반복 키워드 소진 | 같은 꼼수가 실제로 반복될 때만 |
| [ ] | 추적 위험, 자유평판, 세계반응 시계 | 출구 직후 또는 첫 바깥밤 |

출구 조건:

```text
필수 생존 체크 8개 + 출구 단서 1개 = 출구 개방
```

## 5. 샘플 2인 파티

처음이면 `미라 + 보리`가 가장 쉽다.

| 캐릭터 | 역할 | 첫 세션에서 할 일 |
|---|---|---|
| 미라 | 도감, 생존, 관찰 | 냄새, 흔적, 먹을 수 있는 것, 약점을 본다 |
| 보리 | 관계, 조련, 침착한 행동 | NPC와 말하고, 동물을 유인하고, 싸움을 우회한다 |

첫 세션 분담:

| 장면 | 미라 | 보리 |
|---|---|---|
| 배식 | 죽 냄새와 굶주린 사람을 살핀다 | 배식 담당이나 노예와 말한다 |
| 훈련 | 모래, 발자국, 무기 상태를 본다 | 조련사처럼 상대 움직임을 읽는다 |
| 첫 경기 | 약점 단서를 찾는다 | 먹이, 소리, 위치로 상대를 흔든다 |
| 밤의 거래 | 방문자의 손과 냄새를 본다 | 방문자에게 말을 걸고 의도를 본다 |
| 출구 | 흔적을 지운다 | 같이 나갈 사람을 고른다 |

## 6. 긴 플레이 예시: 미라와 보리의 탈출

이 예시는 새 룰을 만들지 않는다.  
위 체크 카드와 전투 로그가 실제로 어떻게 쓰이는지 보여준다.

### 시작

PC는 미라와 보리다.  
둘 다 콜로세움의 노예 우리에서 시작한다.

현재 상태:

```text
배고픔 2
피로 최대치 -1
현재 피로는 줄어든 최대치까지 채움
관중 호응 0
주인 의심 0
출구 단서 0
```

### 장면 1: 배식

노예죽 냄새가 우리 안으로 들어온다.  
앞줄의 굶주린 노예가 보리의 그릇을 노린다.

보리: "잠깐. 네가 내 죽을 뺏으면, 저 뒤에 있는 애들은 다음 차례에 널 노릴걸."

AGM은 NPC 의도를 굴린다.

```text
성격: 생존자
감정: 굶주림
기억: 같은 처지로 봄
겉의도: 식량 확보
```

보리는 매력+설득으로 판정한다.  
상황 슬롯은 `같은 처지라는 말`로 +1d를 받는다.

결과:

```text
보리는 죽을 지킨다.
굶주린 노예는 물러나지만 보리의 얼굴을 기억한다.
미라는 죽 냄새에서 상한 기름 냄새를 맡고, 다음 배식이 더 나빠질 수 있음을 안다.
```

체크:

```text
[x] 판정 슬롯
[x] 배고픔, 음식, 피로
[x] NPC 성향과 대화
```

### 장면 2: 훈련

늙은 검투사가 붉은 모래를 발로 찬다.

"살고 싶으면 멋있게 싸우지 마라. 먼저 살아."

미라는 모래 위 발자국을 본다.  
보리는 먹이 냄새가 나는 낡은 끈을 챙긴다.

미라가 지성+감각으로 훈련 판정을 한다.  
6이 2개 나와 기회 1개를 얻는다.

결과:

```text
미라는 첫 경기에서 상대 이동을 한 번 읽을 수 있다.
기회 1개는 첫 경기에서 적 방어 -1d로 쓰기로 기록한다.
보리는 낡은 끈을 상황 이점으로 준비한다.
```

체크:

```text
[x] 유리/불리, 6의 기회, 꼬임
```

### 장면 3: 첫 경기

상대는 귀족의 애완 주닭이다.  
작지만 사납고, 관중석을 향해 목을 세운다.

전투 로그:

| 항목 | 기록 |
|---|---|
| 라운드/거리 | 1라운드, 가까움 |
| 현재 압력 | 관중이 웃는다 |
| 적 AI | 먹이 냄새가 나는 대상에게 돌진 |
| 플레이어 선언 | 보리가 낡은 끈으로 먹이 방향을 속인다 |
| 공격/방어 결과 | 보리 성공, 주닭 이동 제한 |
| 피해 = 무기 + 순성공 - 방호 | 피해 없음, 상황 이점 |
| 버티기 사용 여부 | 없음 |
| 장비 내구 변화 | 낡은 끈 내구 -1 |
| 부상/상태 이상 | 주닭 `속박 1단계` |
| 장면 여파 | 관중 호응 +1 후보 |

미라는 도감 판정을 시도한다.  
지성+도감으로 주닭의 습성을 본다.

결과:

```text
미라: "저건 사람을 무는 게 아니라, 흔들리는 걸 먼저 문다."
보리는 끈을 던지고 뒤로 빠진다.
주닭은 끈을 물고 한 박자 늦는다.
```

다음 라운드에 미라가 단검으로 위협하고, 보리가 소리로 몰아간다.  
주닭은 쓰러지지 않았지만 이동이 막혀 경기 판정은 PC 쪽 승리로 끝난다.

체크:

```text
[x] 거리, 공격, 방어
[x] 적 AI와 현재 압력
[x] 방호, 내구, 부상 또는 상태 이상
[x] 도감 판정과 정보 슬롯
```

### 장면 4: 밤의 거래

밤에 조련사 견습이 우리 뒤로 온다.  
그는 보리에게 낮에 주닭을 죽이지 않은 이유를 묻는다.

보리: "짐승은 겁먹으면 물어. 사람도 그렇고."

AGM 질문:

```text
저 사람의 말투, 손동작, 시선 중 앞뒤가 안 맞는 것은 무엇인가?
```

답:

```text
그는 보리를 보며 말하지만, 손은 계속 열쇠 꾸러미가 아니라 장부 조각을 만진다.
```

미라는 장부 조각의 냄새를 맡는다.  
먹물과 짐승 기름이 섞여 있다.

조련사 견습은 말한다.

"새벽에 호송마차가 나간다. 너희가 사슬을 끊고 싶은 거라면 문은 못 연다. 하지만 짐승 우리 쪽 바퀴는 낡았다."

보리는 낡은 끈을 고쳐 임시 도구로 만든다.  
수리 판정은 성공한다.

체크:

```text
[x] AGM 질문
[x] 수리, 제작, 개조 중 하나
```

출구 단서:

```text
호송마차의 짐승 우리 쪽 바퀴가 낡았다.
```

### 장면 5: 출구

필수 생존 체크 8개는 끝났다.  
출구 단서도 있다.

심화 체크 중 마법/율법/화신, 반복 보상, 세계 시계는 아직 남았다.  
하지만 억지로 넣지 않는다. 첫 바깥밤으로 넘긴다.

출구 방식은 `다른 도시로 호송 중 탈주`로 정한다.

새벽, 호송마차가 모래길을 지난다.  
보리가 고친 끈이 바퀴에 감긴다.  
미라는 발자국을 숨기기 좋은 낮은 도랑을 본다.

결과:

```text
마차가 기울고 짐승 우리가 열린다.
미라와 보리는 사슬 흔적을 남긴 채 도랑으로 굴러 떨어진다.
뒤에서는 경비가 욕하고, 앞에서는 아직 판데아의 길이 보이지 않는다.
```

### 첫 바깥밤

밤이 온다.  
자유는 안전이 아니다.

DAGM 세이브:

```text
현재 장소: 라피아 외곽 마른 도랑
현재 시간: 탈출 첫 밤
PC 상태: 배고픔 2, 피로 최대치 -1(현재 피로는 최대치), 사슬 흔적
공개 단서: 호송마차 바퀴 파손, 장부 조각 냄새, 주닭을 살려둔 일
숨은 장부 요약: 조련사 견습은 장부 조각 때문에 PC를 골랐다
세계 시계: 추적 위험 +1
감정선: 보리는 짐승을 죽이지 않은 선택으로 조련사 견습에게 기억됨
다음 세션 첫 문장: 흙냄새보다 먼저, 멀리서 주닭 울음소리가 들린다.
```

첫 바깥밤에 처리할 수 있는 이월 체크:

```text
[ ] 마법/율법/화신 목격
[ ] 보상 소스와 반복 키워드 소진
[x] 추적 위험, 자유평판, 세계반응 시계
```

선택:

```text
1. 도랑을 따라 숨어 간다.
2. 호송마차 잔해에서 물건을 챙긴다.
3. 조련사 견습이 왜 장부 조각을 쥐고 있었는지 추적한다.
```
