입력

질문을 적고 d20 두 개로 답변의 뼈대를 만든다.

응답 카드

종이로 하는 법

이 페이지 없이도 아래 순서대로 굴리면 된다. 귀찮지만 그래서 아날로그다.

  1. 질문에서 핵심 단어 1~3개를 동그라미 친다.
  2. d20 두 개를 굴린다. 첫 번째는 질문 읽기, 두 번째는 답변 생성이다.
  3. 합계는 답변 밀도, 차이는 꼬임을 정한다.
  4. 높은 값은 판데아 색, 낮은 값은 검증 규칙을 정한다.
  5. 표 결과를 그대로 쓰지 말고, 질문에 맞게 바꿔 한 문단으로 조립한다.

2d20 표

d20 두 개와 파생값으로 보는 표다.
